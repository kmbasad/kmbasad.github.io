# সোমবার — হরিত মণ্ডপ

## সোমবারের প্রসঙ্গ চীন কন্যার বিবরণ

| বাংলা | # |
| :-- | --: |
| *(রাগ মেলোয়ার। দোপদী ছন্দ)* |  |
| সোমবার প্রাতে বাহরাম গুণবান।। |  |
| নীলমনি টিঙি যথা চন্দ্র অধিষ্ঠান॥ |  |
| জর্কসিন নীল বস্ত্র শিরে ছত্র নীল।। |  |
| অত্যন্ত হিরেষ নৃপ তথাতে চলিল॥ |  |
| সেই গৃহে চীন নৃপ কন্যা এখলাজ।। |  |
| নিলক দামাষ্ক বস্ত্র করিয়া ভূষাজ॥ |  |
| নীলমনি অলঙ্কার পির সর্ব গায়।। |  |
| ঝলকে নরান মাঝে যুতির প্রভায়॥ |  |
| ভিন্নং বর্ণ সখি এক বর্ণ বাসা।। |  |
| নক্ষত্র মণ্ডলে যেন চন্দ্রিমা প্রকাশ॥ |  |
| নৃত্য গীত বাদ্য যত্র সুসৌরভ অঙ্গে।। |  |
| গৃহের সীমায় আসি দাওয়াইল রঙে॥ |  |
| নৃপতি সহিতে যদি হৈল সম দৃষ্টি।। |  |
| সেবে মিলি করিল স্বগন্ধি পুষ্প রক্তি॥ |  |
| ভুরু ভেঙে দিব্য রঙে নরান নরানে।। |  |
| ভেদিল নৃপতি মর্ম কটাক্ষ সঙ্ঘনে॥ |  |
| প্রবল মদন শর ছেদে প্রবেশিল।। |  |
| অতি ভাবে নৃপতি আপনা পাসরিল॥ |  |
| দিব্য ভাব যুক্তিলাভ আপ্ত বিশ্মরণ।। |  |
| অপ্রত্যয় সত্য তত্ত্ব ভাবের স্বপন॥ |  |
| বিজ্ঞাবালা বিস্তোর দেখিয়া নরপতি।। |  |
| প্রণামিয়া করেতে ধরিল শীঘ্রগতি।॥ |  |
| বেক্ত দেখ তত্তাতত্ত ভাবের প্রমাণ। |  |
| শীঘ্রারতি পাত্র নাসি মানিনীর মান।॥ |  |
| দিব্য ভাব সত্ত্বের মানস হয় সিদ্ধি। |  |
| দরিদ্রের হন্তে বিধি মিলাইল নিধি।॥ |  |
| হাস্যোল্লাসে নৃত্য গীতে গৃহে প্রবেশিল। |  |
| সমস্ত দিবস সুখ রঙে নির্বাহিল।॥ |  |
| স্বর্ণ হংস ডুব যদি দিল সিন্ধু নীলে। |  |
| উপরিজ্জিত বিন্দুকুল তাহার হিল্লোলে।॥ |  |
| রসভোগে রজনী অর্ধেক নির্বাহিল। |  |
| শয়ন সময় রাজা কন্যাকে কহিল।॥ |  |
| কহ গুণবতি এক দিব্য সুপ্রসঙ্গ। |  |
| শুনিতে হৃদয় হোক আনন্দ তরঙ্গ।॥ |  |
| কহিলা মোহিনী বালা ভূমি চুষ দিয়া। |  |
| আয়ু যশ বৃদ্ধি হোক শত্রু বিনাশিয়া।॥ |  |
| রূপ মন বশ হেন নজানি কথন। |  |
| নপারি ঈশ্বর আজ্ঞা করিতে লংঘন।॥ |  |
| তেকারণে প্রকাশিমু কথা মনোগত। |  |
| জ্ঞানবন্তু লোক মুখে শুনেছি যেমত।॥ |  |
| উত্তম পুরুষ এক ছিল রূম দেশে। |  |
| বসর তাহার নাম সর্ব লোকে ভোষে।॥ |  |
| বহিবধ গুণ ধরে মোহন পণ্ডিত। |  |
| পুণ্য কর্মে নীতি ধর্মে দেশ প্রতিষ্ঠিত।॥ |  |
| কদাচিত কুকর্মে না ছিল তার মতি। |  |
| ক্ষেমাশীল বসর হইল তার কৃতি।॥ |  |
| একসন্ধ্যা ভক্ষ যদি থাকে তার ঘরে। |  |
| আপনে নখাই তাহা দেয় ভিক্ষুকেরে।॥ |  |
| অতি জ্ঞানবন্তু প্রভু সেবায় তৎপর। |  |
| মহাজন সেবে করে বহুল আদর।॥ |  |
| একদিন ক্ষেমাশীল বসর স্থমিত। |  |
| কার্য্য হেতু গৃহ হন্তে কির ছিল গতি।॥ |  |
| ঈশ্বরের ভাব যেই ঈশ্বর ভাবে লীন। |  |
| কিবা গম্য বিশ্রামে ভাবয় নিশি দিন।॥ |  |
| পন্থ অনুসারে এক পরম সুন্দরী। |  |
| বাসে অঙ্গ ঢাকি আইসে সখি সঙ্গে কির।॥ |  |
| মুখ পরে বোরকা পাছুকা দুই পায়। |  |
| আপাদ মস্তক আদি দেখন নজায়।॥ |  |
| সেই ক্ষণে আচিষ্টে দৈব নিয়োজন। |  |
| পবনে উড়াইল তার মুখের বসন।॥ |  |
| যেন অত্র হন্তে নিষ্টরিল পূর্ণ চন্দ্র। |  |
| দেখিয়া বসর মুখ আঁখি হৈল ধন্দ।॥ |  |
| বিধির নির্বন্ধ দোহ হৈল সম দৃষ্টি। |  |
| মোহিনীর অভ্যাস সতত শর বৃষ্টি।॥ |  |
| ভুরুর ভঙ্গিমা আঁখি বঙ্কিম চালনী। |  |
| মৃদু হাসি সুধারাসি ধরে শুলঙ্কনী।॥ |  |
| একেত লাবণ্য স্বর মুনি মন হরে। |  |
| সেই ভঙ্গি সঙ্গি লঙ্ঘে সহয় সত্ত্বরে।॥ |  |
| বসরের হৃদয়ে লাগিল পঙ্ক বাণ। |  |
| হারাইল ক্ষেমাশীল ধৈর্য্য বুদ্ধিমান।॥ |  |
| হৈয়া স্তব্ধ আহা শব্দ নিধন নির্বাস। |  |
| ভূমে পড়ি দড়মড়ি আছে মাত্র শ্বাস।॥ |  |
| তাহা দেখি চন্দ্রমুখি হৈয়া সলজ্জিত। |  |
| বস্ত্রে মুখ ঢাকি বালা চলিল তুরিত।॥ |  |
| কুল লাজ স্মরি বালা হইয়া মগদ। |  |
| তুরিত গমনে গেলা নিচিন্তিয়া বদ।॥ |  |
| কতক্ষণে বসরের চেতন প্রসারিণা। |  |
| সত্য স্বপ্ন দেখিয়া রহিল ধন্দ হৈয়া।॥ |  |
| মনে ভাবে যদি তার পাছে পাছে জাম। |  |
| ক্ষেমাশীল বেলে লোকে লাজের ভ্রাম।॥ |  |
| ধৈর্য্য ধরিবারে নারি হৈলুম হীন শক্তি। |  |
| চঞ্চল হইলে কভু নপাইয়ু যুক্তি।॥ |  |
| গুপ্ত চেষ্টা কুকর্ম সহজে ধন হীন। |  |
| অন্যে খণ্ডাইতে নারে আমার কুদিন।॥ |  |
| যে মোরে করাইল এই রূপ দর্শন। |  |
| তাহাতে মাগিতে যুক্ত হয় ধীর মন।॥ |  |
| দড় ভাবে ঈশ্বরত মাগিতে বাঞ্ছিত। |  |
| বয়তুল মোকদ্দছে চলিল তুরিত।॥ |  |
| ভক্ষ জল পেয়ের লইল চেষ্টা করি। |  |
| নিষ্টরিল চিতে এক ভাব দড় করি।॥ |  |
| বিশ্রাম করিতে নারে মনেত হতাশ। |  |
| নিশি দিশি চলি গেল প্রভু গৃহ পাস।॥ |  |
| সপ্তবার প্রদক্ষিণ করি সেই ঘর। |  |
| দড় চিতে দড় ভাবে মাগিলেক বর।॥ |  |
| দীনবন্ধু দয়া সিন্ধু মাত্র তুমি সার। |  |
| মন ইচ্ছা সব মিছা গোচরে তোমার।॥ |  |
| তোমার সৃজন আদি ত্রিভুবন যত্র। |  |
| বিনা তোমা আজ্ঞায় না লড়ে এক পত্র।॥ |  |
| যদি তোমা মতি নেহ সুন্দর শরীরে। |  |
| ভাবকের চিত্ত আর কে হিরেতে পারে।॥ |  |
| ভাবকের চিতে প্রেমানল জ্বালাইয়া। |  |
| আপনে হরহ দিব্য মুরতি ধরিয়া।॥ |  |
| যেই রূপ দর্শাইলা মোহর নয়ানে। |  |
| তুমি মাত্র সান্তদাতা সেই রূপ দানে।॥ |  |
| মনবাঞ্ছা প্রাপ্তি হেতু লক্ষ্য নাহি আর। |  |
| ভক্তি ভাবে লৈলুম নাম স্মরণ তোমার।॥ |  |
| বাঞ্ছা দান দিতে পার তুমি সব কর্তা। |  |
| নেহ ক্ষেমা ধৈর্য্য দেও এই দুঃখ হর্তা।॥ |  |
| বহিবধ প্রণামী প্রভুতে মাগি বর। |  |
| তথা হন্তে উদ্দেসি চলিল নিজ ঘর।॥ |  |
| পন্থক্রমে একজন সংহিত মিলিল। |  |
| দেখিতে সভাতাশীল প্রকৃতি কুটিল।॥ |  |
| প্রতি শব্দে ছল গ্রাহি বাক্য যুদ্ধ করে। |  |
| অন্য কি সে পরম ঈশ্বর ছিদ্র ধরে।॥ |  |
| যখনে বসরের যুক্তি বাক্য প্রকাশয়। |  |
| অনুচিত কহি তারে বিরূপ বোলয়।॥ |  |
| এক বাক্যে দেয় তারে সত পদুত্তর। |  |
| মন্দ ভালা আলা ঝালা বোলয়ে বিস্তর।॥ |  |
| বসরে দেখিয়া তার চরিত্র কুচ্ছিত। |  |
| কর্ণ ব্যাজা বৈকল বাক্য তেজি মৌন রিত।॥ |  |
| সেতক বচনে অক নন্দে পদুত্তর। |  |
| ফিরে ফিরে কহে তবে বসর গোচর।॥ |  |
| তবে বসরেত জিজ্ঞাসিল পুনর্বার। |  |
| কহ শুনি গুণমনি কি নাম তোমার।॥ |  |
| আপনার নাম যদি বসরে কহিল। |  |
| সেই ক্ষণে পুনরপি কহিতে লাগিল।॥ |  |
| বলিল উত্তম নাম লক্ষণ সুচারু। |  |
| মল্লিকা আমার নাম জগতের গুরু।॥ |  |
| নানা শাস্ত্র পড়িয়া প্রবল হৈল বুদ্ধি। |  |
| স্বর্গ মর্ত্ব পাতালের সব জানি শুদ্ধি।॥ |  |
| যত বস্তু আছে সিন্ধু পর্বত কানন। |  |
| সকলের মন মর্ম আছে মোর মনে।॥ |  |
| যার যেই উৎপত্তি প্রলয় যেই মত। |  |
| চিতের মুকুরে মোর সকল ব্যাকত।॥ |  |
| স্বর্গ তারা ঋতি ধারা পারেঁ গনিবার।। |  |
| ভাল মন্দ নানা ছন্দ আগে জানেঁ তার॥ |  |
| নৃপতির রাজ্য ভঙ্গ হৈব যেই মতে।। |  |
| পঞ্চাশ বৎসর আদ্যে আমার বিদিতে॥ |  |
| শস্য দ্রব্য সঙ্গে মার্গি হইব যেন ঋদ্ধি।। |  |
| বৎসরক আগে জানি তাহার যে শুদ্ধি॥ |  |
| কষ্ট আদি ব্যাধি যত কার্য্যের অনর্থ।। |  |
| তিলে পল টাইতে পারি অধিক সমর্থ॥ |  |
| নর আদি যতেক জন্তুর নানা ব্যাধি।। |  |
| ফুকেত আরোগ্য করি কি কাম ঔষধি॥ |  |
| এক ফুক দেওঁ যদি জ্বালিয়া আঙটিন।। |  |
| প্রতি ধাওঁ হেম রত্ব হয় কুটামিন॥ |  |
| ভূমি শিলে রক্ব যত ব্যাধির নির্মাণ।। |  |
| আমাতে ব্যক্ত সব যে আছে যেই স্থান॥ |  |
| স্বর্গ মর্ত্ব পাতালের যত গুপ্ত কথা।। |  |
| জিজ্ঞাসিলে পারি মাত্র দিতে তার বার্তা॥ |  |
| নানা স্থানে যতেক দেখিছি গুণবন্তু।। |  |
| মোর ধিক উপাসক নাহিক মোহন্তু॥ |  |
| কেহ দুই কেহ চারি বিদ্যা মাত্র জানে।। |  |
| গুরু শ্রেষ্ঠ গুরু আমি দেখ সর্ব স্থানে॥ |  |

| চিরকাল আনন্দে গোঁয়াইল দুই জন। |  |
| যথা ধ্যান তথা লাভ বিধি নিয়োজন॥ |  |
| যদি চিন নৃপ কন্যা এখেলাজ নাম। |  |
| এই কথা কহিল শুনিল বাহরাম॥ |  |
| নানাবিধ রত্ন অলঙ্কার বস্তু দিয়া। |  |
| শয়ন করিল কন্যা বক্ষে লাগাইয়া॥ |  |
| বৃক্ষ আদি বনস্পতি নীলবর্ণ ধরে। |  |
| নীলবর্ণ নয়ানের যুতি ধিক করে॥ |  |
| ফিরিস্তা সবুজ বর্ণ জাহিদ ফকির। |  |
| পক্ষী মধ্যে সুপিণ্ডিত নীলবর্ণ কীর॥ |  |
| ধান্য আদি শস্য যত জীব রক্ষাকারী। |  |
| অঙ্কুর হইতে সব নীল বর্ণধারী॥ |  |
| বহু মূল্য ধরে যুতিমন্ত নীলমণি। |  |
| নীলবর্ণ দেব হিন্দু দেব শ্রেষ্ঠ মানি॥ |  |
| শ্রীযুত ছৈয়দ মহাম্মদ গুণবান। |  |
| ভুবন ভরিয়া যার কীর্ত্তির বাখান॥ |  |
| এই পরস্তাব শুনি অন্তর হরিষে। |  |
| হীন আলাওল বাক্য অমিয়া বিরেষে॥ |  |
