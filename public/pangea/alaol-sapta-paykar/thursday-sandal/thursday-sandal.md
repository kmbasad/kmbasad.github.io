# বৃহস্পতিবার — চন্দন মণ্ডপ

## বৃহস্পতিবারের প্রসঙ্গ হর পরী কন্যার বিবরণ

| বাংলা | # |
| :-- | --: |
| *(রাগ দৌপদী ছন্দ)* |  |
| বৃহস্পতি বারে বাহরাম মহারাজ। |  |
| চন্দন বর্ণ গৃহে যাইতে কেল সাজ॥ |  |
| চন্দন বর্ণ গৃহ গুরু অধিষ্ঠান। |  |
| সেই বর্ণ বস্ত্র ছত্র তথাত পরান॥ |  |
| সেই গৃহে বৈসেস কন্যা হর পরী নাম। |  |
| কেয়ানি বংশের কন্যা অতি অনুপাম॥ |  |
| গৃহ বর্ণ বস্ত্র অলংকার পিরি অঙ্গে। |  |
| গৃহের সীমায় আসি সখিগণ সঙ্গে॥ |  |
| নৃপ দরশনে হেয়া অধিক হরিষ। |  |
| করিল চন্দন আদি সৌরভ বিরষ॥ |  |
| মোহিয়া নৃপতি মন কটাক্ষ সন্ধানে। |  |
| বিভোর হইতে করে ধরিল আপনে॥ |  |
| নৃত্য গীত উৎসব আনন্দ সবিশেষ। |  |
| গৃহের অন্তরে নিয়া করাইল প্রবেশ॥ |  |
| সমস্ত দিবস বহি গেল নানা রঙ্গে। |  |
| উগিল চন্দ্রিমা নিশি তারাগণ সঙ্গে॥ |  |
| কেলি রসে অবশেষে বলিল রাজন। |  |
| কহ গুণবতী এক উত্তম কথন॥ |  |
| প্রণামিয়া আশীর্বাদ শেষে রাজ রামা। |  |
| বলে নৃপ মন বশ না জানি উপমা॥ |  |
| তবে কি ঈশ্বর আজ্ঞা লংঘিতে না পারি। |  |
| তেকারণে মনোগত কহিমু প্রচারি॥ |  |
| মোর দেশে পূর্বকালে ছিল দুই ইষ্ট। |  |
| শিষ্ট নাম একজন দ্বিতীয় অশিষ্ট॥ |  |
| যার যেন মত নাম তেমত চরিত। |  |
| তথাপি সাহায্য ভাবে দুই ছিল মিত॥ |  |
| কার্য হেতু সঙ্গতি চলিল দূর ভুমে। |  |
| পথের সম্বল জল কেলা অনুক্রমে॥ |  |
| বলবন্ত অশিষ্ট যে লইল পূর্ণিত। |  |
| মৃদু তনু শিষ্টে বহি কেল যথোচিত॥ |  |
| অশিষ্টে জানয় আগে সেই পথ চিন। |  |
| সপ্ত দিন বাটমধ্যে জল আছে হীন॥ |  |
| শিষ্ট স্থানে অশিষ্ট কহিল মন কম্প। |  |
| তোমা সঙ্গে সিলল সম্বল মাত্র অল্প॥ |  |
| তাহারে ভিক্ষিয়া আগে যাও সহসাত। |  |
| বহু দ্রব্য অনুচিত আগে ব্যয় হাত॥ |  |
| শিষ্ট বলে পথের মরম জান তুমি। |  |
| তোমা আজ্ঞা পালিয়া চলিছি সঙ্গে আমি॥ |  |
| তোমা সঙ্গে দুর্গম পথেত চলি যাইব। |  |
| যেই মত আজ্ঞা কর তেমত চলিব॥ |  |
| এই মেত কত দিন পথে চলি গেল। |  |
| শিষ্ট সঙ্গে সিলল সমস্ত সাংহেল॥ |  |
| দূর ভুমি গিয়া তবে হেল উপস্থিত। |  |
| সপ্ত দিন জল হীন পথ সে ভুমিত॥ |  |
| তৃষাকুল হেয়া শিষ্ট মাগিলেক নীর। |  |
| বলিলেক ক্ষেণেক চলহ ধীরে ধার॥ |  |
| বলবন্ত অশিষ্ট গমন শীঘ্রে যায়। |  |
| তৃষাকুল হেলে গিয়া দূরে জল খায়॥ |  |
| হেচে তপ্ত বালু উর্দ্ধে অরুণ প্রচণ্ড। |  |
| চলিতে না পারে শিষ্টে মর্ম খণ্ড খণ্ড॥ |  |
| কাকুতি করিয়া বলে আহে পুণ্যবান। |  |
| কিছু জল দিয়া রাখ আমার পরাণ॥ |  |
| ক্রোধ হেয়া অশিষ্টে করয় বিষসাদ। |  |
| বলে জল কিনি খাও যদি থাকে সাধ॥ |  |
| শিষ্টে বলে ধন প্রাণ সকল তোমার। |  |
| তরাইয়া লহ তুমি দুর্গম উদ্ধার॥ |  |
| দুই রত্ন শিষ্ট স্থানে আছে বহু মূল। |  |
| বলে ইহা লই মোরে কর অনুকূল॥ |  |
| অশিষ্টে বলয় তারে কেয়া কোন্ কাজ। |  |
| মনুষ্য সমাজে মোরে দিবে মহা লাজ॥ |  |
| শিষ্টে বলে যেন মেত তোমার প্রত্যয়। |  |
| তেমত শপথ মোরে দেহ মহাশয়॥ |  |
| অশিষ্টে বলয় মোকে না লাগে সে বাজি। |  |
| এ সব শেতক ছল আমি সব বুঝি॥ |  |
| যদি তোর আরতি করিতে জলপান। |  |
| খসাইয়া দেও মোরে যুগল নয়ান॥ |  |
| শিষ্টে বলে অধিক জীবন মৃত্যু মূল। |  |
| কুমরণ যে মরে হইয়া তৃষাকুল॥ |  |
| মেনে ভাবে প্রাণাধিক না হয় নয়ন। |  |
| কি কর্ম নয়ন যদি নাং রহে জীবন॥ |  |
| মোর আঁখি রত্ন লও আয় পুণ্যবান। |  |
| মাত্র এক তৃষাপূর্ণ জল দেও দান॥ |  |
| তৃষাপূর্ণ জল যদি পিলাও আমারে। |  |
| সব অপরাধ আমি ক্ষেমিব তোমারে॥ |  |
| ঈশ্বর আদেশ বিনা না দোলয় পাতা। |  |
| আপনা নিমিত্তে মাত্র ভাল মন্দ যথা॥ |  |
| এত শুনি শীঘ্র করাইয়া জলপান। |  |
| তীক্ষ্ণ ছুরি হানি কেল যুগল নয়ান॥ |  |
| নিবাইতে প্রদীপ না কেল মনে শঙ্কা। |  |
| হেন কর্ম না করে বসতি যার লঙ্কা॥ |  |
| জ্যোতি স্থুল হেতে নিষ্পরয় জলধার। |  |
| কর্ম ভাবি রহিলেক স্থির করতার॥ |  |
| অশিষ্টে তাহার ধন রত্ন সব কেলা। |  |
| হিরেষ চলিল পথে অগ্রগামী হেয়া॥ |  |
| দণ্ড চারি অন্তরে আছয় এক গ্রাম। |  |
| তথা এক মোহন্ত পুরুষ গোখ নাম॥ |  |
| উট গাভী মেষ ছাগ আছয় বহুল। |  |
| যথা তৃণ আছয় চরায় পশুকুল॥ |  |
| সেই দিব্য স্থানে তৃণ সব ভরপুর। |  |
| মিষ্ট জল ঝরনা আছয় কতদূর॥ |  |
| সেই স্থানে আইল গোখ পশুকুল সঙ্গে। |  |
| দুহিতা সুন্দরী তার চিল আইল রঙ্গে॥ |  |
| ভুবন মোহন কন্যা নবীন বয়সী। |  |
| রূপ দেখি লজ্জা পায় পূর্ণিমার শশি॥ |  |
| মোহনী চতুরী বালা দয়াল হৃদয়। |  |
| নীতি ধর্ম কর্মে বালা সরস হৃদয়॥ |  |
| দিব্য জলে অঙ্গ পাখালিতে শ্রদ্ধা হেয়া। |  |
| ঝরনা নিকেটে গেল উটে আরোহিয়া॥ |  |
| পঞ্চম মোসক তুলি কেলা উট পরে। |  |
| গোছল করিয়া কন্যা আইসে ধীরে ধীরে॥ |  |
| চেক্করের বেদনায় শিষ্টে প্রান্তরে পড়িয়া। |  |
| উচ্চৈঃস্বরে কান্দে দীনবন্ধুকে স্মরিয়া॥ |  |

## শিষ্ট গোখের কন্যা হেতে চক্ষুদান পাইবার এবং তাহার কন্যার সঙ্গে বিবাহ হইবার বিবরণ

| বাংলা | # |
| :-- | --: |
| *(রাগ জমক ছন্দ)* |  |
| দুঃখের ক্রন্দনে হয় পাষাণ বিদার। |  |
| স্ব-রুধিরে আখি নীর বহে অনিবার॥ |  |

| কাতর হইয়া কান্দে লই প্রভু নাম॥ |  |

| শান্তাইয়া জিজ্ঞাসিল তুমি কোন জন। |  |

| কোন্ দুষ্টে পাইল এথা কোথা হেতে যাও। |  |

| শিষ্টে বলে কেমন দেবতা আইল এথা। |  |
| অনুমানে বুঝি আমি অরণ্য দেবতা॥ |  |

| বলে ক্ষমা ধরি রহ ঈশ্বর ভাবিয়া॥ |  |
| সুখ শেষে দুঃখ দেয় দুঃখ শেষে সুখ। |  |
| আনে কি করিব সব ঈশ্বর কৌতুক॥ |  |
| এবে তুমি চলি আইস আমার আলয়। |  |
