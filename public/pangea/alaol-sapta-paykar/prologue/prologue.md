# প্রস্তাবনা

## প্রভুর স্তুতি :: Praise of the Lord

| বাংলা | ইংরেজি | # |
| :-- | :-- | --: |
| আদ্যের অনাদি স্বামী অন্তের অনন্ত। | Lord without beginning, infinite beyond all end. |  |
| প্রথমে মহিমা তান সুশোভিত গ্রন্থ॥ | First, may His glory grace this book. |  |
| বিনা লক্ষে শূন্য পরে স্থাপিছে আকাশ। | Aimless, He set the sky upon the void. |  |
| করিছে মিহির শশি নক্ষত্র প্রকাশ॥ | He made the sun, the moon, the stars. |  |
| সকলের কর্তা আপে করিছে শুশম। | Doer of all, He keeps their order. |  |
| ভাঁতিং কার্যগতি রাখিছে নিয়ম॥ | In all ways He holds deeds to rule. |  |
| আজু হেন্তে জীবদান যত চরাচর। | From this day He gives life to all that moves and is still. | 7 |
| তার বলে আয়ুর জীবন নিরন্তর॥ | By His power life runs on unceasing. |  |
| দৃষ্টার দৃশ্যার পরে তার দিব্য জ্যোতি। | On seen and seer His light rests. |  |
| শ্রোতার শ্রবণ মাঝে সেই দিছে শ্রুতি॥ | In the hearer He sets the power to hear. |  |
| সর্ব ভূতে ব্যাপিত সংখ্যাতীত মহিমা। | His glory fills all beings, past counting. |  |
| অতুল মহিমা তার দিতে নারি সীমা॥ | His matchless glory — I can set no bound. |  |
| ত্রিভুবনে নাহি আর দ্বিতীয় সৃজন। | In the three worlds, no second creator. |  |
| ষট রেখা পাল্য মাল্য স্বামী সর্বজন॥ | Lord of all, garlanded with the six directions. | 14 |
| তাহা হেতে নিষ্পরিছে শরীর সবার। | From Him issue the bodies of all. |  |
| পুনরপি তথা সকলের অনুসার॥ | And back to Him all return. |  |
| ভাবিতে চরিত্র তার বুদ্ধিবন্ত বদ্ধ। | Pondering His nature, the wise stand helpless. |  |
| বুঝিতে মরম তান হয় অন্ধ ধন্দ। | Seeking His secret, one falls blind with wonder. |  |
| জেতক জীবন হেছে সৃজন তাঁহার। | Whatever lives is His creation. |  |
| সংসার অসার জান সেই মাত্র সার॥ | The world is hollow; He alone is essence. |  |
| জ্ঞান এক কণা হেন্তে পথ দর্শায়। | From one grain of His knowledge, the path. | 21 |
| কৃপা রূঢ় হেন্তে কর্ম সুশম করায়॥ | By His grace He sets all deeds right. |  |
| তারে দণ্ডবৎ না করিল যেই শিরে। | Whoever bowed no head before Him — |  |
| কুলুপ লাগিল তার মুক্তির দুয়ারে॥ | for him the door of release is locked. |  |
| শ্রোত জ্যোতি রাখি তম শিশির নিকট। | He keeps light near darkness and dew. |  |
| এক না পাসরি ভেক্ষ দেয় প্রতি ঘট॥ | Forgetting none, He gifts every vessel. |  |
| কারুণের গঙ্গ আনি নিলক্ষে লুটায়। | He pours the Ganges of mercy, aimless. |  |
| ভিক্ষুকেরে দেয় নিধি কে তারে বুঝায়॥ | He gives the beggar treasure — who can fathom Him? | 28 |
| পড়ি গুনি জ্ঞানবেন্তে ভাবয় তাঁহারে। | The learned, reading and counting, ponder Him. |  |
| যে তাঁকে জানিল দৃড় সকল পাসরে॥ | Who knows Him firmly forgets all else. |  |
| অগ্নিময়ী শিলা দাহ তন্ত্র জন্ত্র রিতে। | By tantra and mantra the fiery stone burns. |  |
| সকলের সুখ এক ঈশ্বরের ভিতে॥ | The joy of all rests in the one Lord. |  |
| যে সেবে ঈশ্বর শ্মরে আছে একজন। | Who serves and recalls the Lord, standing alone — |  |
| সে করে সকল কর্ম আলগ আপন॥ | performs all his deeds free and his own. |  |
| তাহা হেন্তে সকল জীবন অধিকারী। | From Him alone all beings get their share. | 35 |
| অনন্ত অপার লীলা বুঝিতে না পারি॥ | His endless play — I cannot grasp it. |  |
| সেই সে করিছে সুর শশির প্রকট। | He makes the gods and the moon appear. |  |
| ধবল শ্যামল জ্যোতি দুই অন্তঃস্পট॥ | White light, dark light — both shine within. |  |
| তার গ্রন্থ অবিশ্রাম সতত শুনেন্তু। | His book — may one hear it without rest. |  |
| তাঁহার আদেশ বিনা কিছু না বুঝেন্তু॥ | Without His command, let nothing be known. |  |
| হৃদয়ের মাঝে সেই করিছে উজ্জল। | He lights the heart within. |  |
| তাঁহার দাতব্যে সব পায় বলাবল॥ | By His gift all gain strength. | 42 |
| সেই রতু অনুরূপ সবান বড়াই। | That jewel — to it all greatness fits. |  |
| না জানে তাহার কর্তা আছে কোন ঠাই॥ | None knows where that maker dwells. |  |
| জীবন স্বরূপে প্রতি ঘটে অধিকারী। | As life, He rules over every vessel. |  |
| কোন স্থানে থাকে সেই চিনিতে না পারি॥ | Where He dwells, I cannot tell. |  |
| অঙ্গবাসী নিজ রতু না পারে চিনিতে। | The jewel within the body — none knows his own. |  |
| যে তাকে সৃজিছে তাঁকে চিনিবে কি মতে॥ | how then know the one who made him? |  |
| জীবকর্তা জীব হেতে আছে যে নিকট। | The maker of life is near the living. | 49 |
| জ্ঞানবেন্তে কৃপা হেন্তে জানয় প্রকট॥ | By wisdom and grace He shows to knowing. |  |
| সেই পথ দর্শেকরা অতুল দেখয়। | Who shows that path sees the matchless. |  |
| স্থুল বিবর্জিত মাত্র আছে সর্বময়॥ | Free of all grossness, He fills all. |  |
| যে সব নক্ষত্র ভাবি শুভা-শুভবিরত। | Those who read the stars for good and ill — |  |
| প্রভুর শক্রতা সেই না পারে বুঝিত॥ | cannot grasp the Lord's power. |  |
| যাহারে বলয় ভাল তার মন্দ হয়। | What he calls good turns bad. |  |
| ভিক্ষুকেরে দেয় নিধি কে তাকে বুঝায়॥ | He gives a beggar treasure — who can fathom Him? | 56 |
| পড়ি গুণি জ্ঞানবেন্তে ভাবয় তাহারে। | Reading and counting, by wisdom one meditates on Him. |  |
| যে তাকে জানিল দৃড় সকল পাসরে॥ | Who knows Him firmly forgets all else. |  |
| ত্রিভুবনে যেতক সৃজিল ভাঁতি তাঁতি। | All He made through the three worlds, so varied — |  |
| নিয়ম করিতে নারি ভাবি এক রাতি॥ | I could not count it in a whole night. |  |
| বুদ্ধিবন্ত পথ যেবা পাইছে তা হেতে। | The path the wise have found thereby — |  |
| ধন্দ হই চাহিতে না পারে তার ভিতে॥ | bewildered, none reach its depth. |  |
| অনেক সৃজন এক ভাবিতে বিভোর। | Lost in seeing the many as one — | 63 |
| চিরজীবি বিধাতার কে পাইবে ওর॥ | who can reach the deathless Creator's end? |  |
| আদি অন্ত স্বর্গ মর্ত্য পাতালের স্থিতি। | Beginning and end, heaven, earth, and hell — |  |
| যোগ্য ভেক্ষ্য দেয় নিত্য নাহিক বিস্মৃতি॥ | He feeds each fitly, day by day, forgetting none. |  |
| এতেক ভাবিয়া মনে হইয়া লজ্জিত। | Pondering all this, I am filled with shame. |  |
| বর মাগি কৃপামূর্ণ পুরাও বাঞ্ছিত॥ | I beg a boon: O gracious one, grant my wish. |  |
| জ্ঞান দানে রাখ প্রভু আপন দুয়ারে। | By the gift of knowing, Lord, keep me at Your door. |  |
| অন্যের দুয়ার আশা খণ্ডাও আমারে॥ | Cut from me all hope of another's door. | 70 |
| শক্তির কৃপান দিয়া কাট মন ঘোর। | With the sword of power, cut my mind's dark. |  |
| তোর মর্ম যদি পাই সব যোগ্য মোর॥ | If I reach Your secret, all is made fit for me. |  |
| তুমি মোর গ্রাহক কান্দিমু কার ঠাই। | You receive me — at whose door else shall I weep? |  |
| তুমি মুক্তি দায়ক মাগিমু কাতে যাই॥ | You give release — where else shall I beg? |  |
| যদ্যপি সংসারে বহু আছয় ভেকত। | Though the world wears many masks, |  |
| নাহিক তোমার আগে তিলেক গোপত॥ | not a seed's worth is hidden from You. |  |
| মনের মানস গুপ্ত নাই তোমা স্থানে। | The mind's secret will is not hidden from You. | 77 |
| তুমি সিদ্ধ কর প্রভু জানহ আপনে॥ | You bring it to pass, Lord; You know it. |  |
| সেই কর্ম উত্তম তোমাতে মাগি যারে। | The work I beg of You is the best. |  |
| যে সব নবীন আমি না কহি তোমারে॥ | What is new to me I need not tell You. |  |
| তোমার দুয়ারে আমি মাগি অব্যাহিত। | At Your door I beg without cease. |  |
| আর যত মনে ভাবি তুমি তার পতি॥ | Of all I think, You are the master. |  |
| শক্তিহীন অল্প বুদ্ধি মুই দুরাচার। | Weak, of little sense, a sinner am I. |  |
| তোমা ভাবি দিতেছি যে সমুদ্রে সাঁতার॥ | Fixed on You, I swim an ocean. | 84 |
| তুমি কৃপা করিলে তরিতে পারি সিন্ধু। | By Your grace, I can cross the sea. |  |
| অনাথের নাথ ওহে তুমি দীনবন্ধু॥ | Lord of orphans, friend of the lowly. |  |
| তব কৃপা বিনে কোন কার্যে নাই মুক্তি। | Without Your grace, no work brings release. |  |
| হৃদয়ে প্রকাশি প্রভু দেও উক্তি শক্তি॥ | Shine in my heart, Lord; give me speech. |  |
| আর কৃপা কর প্রভু দয়াল চরিত। | Show more grace, Lord of mercy — |  |
| মহিমা তোমার শখা গাইতে কিঞ্চিত॥ | that I may sing a little of Your glory. |  |
| আদেয়ত নিরূপ ছিল প্রভু নিরাকার। | At first the Lord was formless, beyond likeness. | 91 |
| চেতন স্বরূপ যদি হইল প্রচার॥ | When His conscious nature appeared — |  |
| অতি ঘোরতর ময় আকার বর্জিত। | dense with dark, without form, |  |
| মহাজ্যোতির্ময় হেল ঈশ্বর ইঙ্গিত॥ | the Lord's sign blazed with light. |  |
| জ্যোতির সমুদ্রে আদ্য হুর মহান্মদ। | In that sea of light, first, the light of Muhammad. |  |
| জগত বিজয়ী হেন্তে পাইল সম্পদ॥ | Through him the world-conqueror gained his fortune. |  |
| স্বপুষ্কর্ণা উদ্যানের আদ্য নবফুল। | First new flower of the golden garden — |  |
| বুদ্ধি বাক্য শিরোমনি ভুবন উজ্জল॥ | crown of wisdom and speech, lighting the world. | 98 |
| সর্গ তাজ হেন্তে নবীকুল ছত্রপতি। | From heaven's crown, sovereign of all Prophets — |  |
| শিরয়ত জান তাঁর প্রভু পাশে গতি॥ | knower of the law, ever moving toward his Lord. |  |
| বিনা পাঠে সর্বশাস্ত্র হইয়া বিদিত। | Unread, all scriptures were known to him. |  |
| আরেসর পরম কর্তা থাকে পৃথিবীত॥ | The throne's supreme lord dwells on earth. |  |
| সেই পুষ্প হেন্তে আদ্য আদম উজ্জল। | From that flower Adam first shone — |  |
| সকল কদর্য পূর্ণ সেই সে নির্মল॥ | full of all worth, utterly pure. |  |
| অন্তে জীববন্তু স্বর্গ জিনি গতাগতি। | At last, heaven's victors come and go — | 105 |
| সর্ব গ্রন্থ নাশি হেলা আর গ্রন্থপতি॥ | all scriptures dissolve; he is their lord. |  |
| তান আজ্ঞা নিরোধ হইল সার ধার। | His command became the one great restraint — |  |
| এক বিন্দু দোলাইতে শক্তি আছে কার॥ | who can shake one drop of it? |  |
| নৃপকুল আদেশের গতে চলে কর্ম। | Kings rule their affairs by his command — |  |
| তান আজ্ঞা শীরে নৃপে বুঝে কার্য মর্ম॥ | they bear it on their heads and grasp their duty. |  |
| নির্ধনী মহত্ত জানি হইয়া তুষ্ট মন। | Knowing the worth of want, content of heart — |  |
| দুই জগ নৃপ হইয়া না ইচ্ছিল ধন॥ | king of two worlds, he sought no wealth. | 112 |
| পরিশ্রমে বিদ্যা লক্ষে নিজ ভুজার্জিত। | By his own arm he earned learning and bread — |  |
| ভিক্ষিলা অতিথ পরিজনের সহিত॥ | he begged beside guests and kin. |  |
| সূর্য জ্যোতি সমান ধবল অন্ধ ছায়া। | White as sunlight, casting no shadow — |  |
| সংসারের কে আছে আর ছায়াহীন কায়া॥ | who else has a shadowless body? |  |
| নৃপকুল আজ্ঞা বিনে জীবন অবধি। | Without kings' command, life meets its end — |  |
| প্রলয় পর্যন্ত সার তান আজ্ঞা বিধি॥ | his law is the essence till the world's end. |  |
| অধোগতি হেল যেই উচ্চ হৈল শির। | Whoever lay low, his head was raised — | 119 |
| যে পড়িল হেন্তে ধরি করিল স্থিরস্থির॥ | the fallen he caught and set firm. |  |
| যেই দুষ্ট ভ্রষ্ট তারে হৈলা সব নষ্ট। | The wicked and corrupt — all theirs was lost. |  |
| যে শিষ্ট উৎকৃষ্ট তারে হৈলা স্পষ্ট পুষ্ট॥ | The good and worthy were nourished and throve. |  |
| যেই না মানিল সলোচনে হইয়া অন্ধ। | Who would not own him, blind by their own eyes — |  |
| অদ্যাপিও না এড়ায় সেবে বলে যন্দ॥ | to this day cannot escape, so the witness says. |  |
| স্বর্গবাসী ফেরেস্তা তাহান আজ্ঞা পাল। | Heaven's angels keep his command — |  |
| তান স্তুতি কহি সেবে গোয়ারন্তু কাল॥ | singing his praise, they pass their days in service. | 126 |
| আনে কি কহিবে যারে আপে কর্তার। | What more can any say, when the Creator Himself — |  |
| কহিছে তোমার লাগি সৃজিনু সংসার॥ | said: for your sake I made the world? |  |
| কে বুঝিতে পারে তান মহিমা প্রচণ্ড। | Who can grasp his vast glory? |  |
| অঙ্গুলি ইঙ্গিতে যাঁর চন্দ্র দুই খণ্ড॥ | At whose finger the moon split in two. |  |

| আনন্দে শুনেন সেবে হই এক মন। | he listens glad, and all grow of one mind. |  |
| আমিহ সভাতে তান থাকি অবিরত। | I too stay in his court without cease; |  |
| অন্ন বস্ত্র দানে আমা পোষেন্তু সদত॥ | with food and clothing he ever keeps me. |  |
| মোর মন রস তান প্রেম রাগ রায়। | My heart delights in his love, passion, and song; |  |
| বিশেষ কহিল মোরে আদর কৃপায়॥ | with special love and grace he spoke to me. | 266 |
| তান সভাসদ থাকি সভাসদ হইয়া। | Among his courtiers, made one of them, |  |
| শাস্ত্র নীতি রস কথা প্রসঙ্গ কহিয়া। | speaking of scripture, counsel, and rasa — |  |
| এক নিশি পণ্ডিত সমাজে মহাশয়। | one night, in the learned assembly, that great one |  |
| কথা রসে বসিছেন্তু আপনা আলয়॥ | sat in his hall, in the savour of talk. |  |
| আমা প্রতি হৈলা আজ্ঞা হরিষত মনে। | To me, with glad heart, came the command, |  |
| উত্তম প্রসঙ্গ এক কহিতে কারণে। | to tell a certain fine tale. |  |
| সপ্ত পয়কর কথা অতি মনোহর। | The Sapta Paykar's tale is most charming — | 273 |
| মনগত প্রকাশিলুং তাহান গোচর॥ | I opened my heart's thought before him. |  |
| যেন মতে নৃপ এক পির বস্ত্র শ্যাম। | How a king, in a dervish's dark robe, |  |
| নিশি দিশি কান্দিয়া গোয়াইল অবিশ্রাম। | wept night and day without rest — |  |
| পশ্চাত কহিমু দেখি না কহিনু এথা। | I tell it later, not here; |  |
| মহা উল্লাসিত হেল শুনি সেই কথা॥ | hearing it, he was greatly gladdened. |  |
| তবে মোরে আদেশিল হাসিতেই। | Then, smiling, he gave the command: |  |
| যত্ন করি এই কথা পয়ারে রিচিতে॥ | to render this tale with care in payar. | 280 |
| পারশ্য আরবী ভাষে এতরাজ ছন্দ। | In Persian and Arabic this metre is etaraj — |  |
| বিশেষ নেজামি বাক্য সাগরে প্রবন্ধ॥ | Nizami's work is an ocean of words. |  |
| এই গ্রন্থ মাঝে আর যত ইতিহাস। | All the other tales in this book — |  |
| পয়ারের প্রবন্ধে তাকে করহ প্রকাশ॥ | set them in payar verse. |  |
| একে মহা পুরুষ বিশেষ পালইতা। | He is a great man, a nourisher — |  |
| পিতার সমান শাস্ত্রে বলে অন্নদাতা॥ | scripture calls the food-giver a father's equal. |  |
| তান আজ্ঞা লংঘিতে না পারি কদাচিত। | I can never cross his command. | 287 |
| যদ্যপিও জরাজীর্ণ চিন্তাকুল চিত॥ | Though my mind is worn with age and care. |  |
| যদিবা অযোগ্য আমি গ্রন্থ রচিবার। | Even if unfit to make a book, |  |
| তান ভাগ্য লক্ষ্যে হয় সমুদ্রে সঁকার॥ | by his fortune's sign one dives the sea. |  |
| যেন চন্দ্র ধরিতে বালক হস্ত তোলে। | As a child lifts his hand for the moon, |  |
| কেবল ভরসা মাত্র গুরু পদতলে॥ | my only trust is my master's feet. |  |
| বিষম সুশম হয় মহন্ত আজ্ঞায়। | The rough turns smooth at the great one's word; |  |
| অন্ধকার জ্যোতি হয় গুরুর কৃপায়॥ | dark turns to light by the master's grace. | 294 |
| এত ভাবি সাহস করিলুং মতি হীনে। | So thinking, I, weak-minded, took heart, |  |
| ভক্তি প্রণতি করি গুরুর চরণে॥ | and bow in devotion at my master's feet. |  |
| নিজ গুণে শোধিও বিচারি পাইলে দোষ। | By your own virtue, weigh and mend each fault. |  |
| বিন অবধানে না হইও অসন্তোষ॥ | Be not displeased without due thought. |  |
| গ্রন্থের গ্রাহক কর্ম ছোট শক্তি নয়। | To read a book with judgment is no small skill; |  |
| মহাজনে তার মাত্র মরম বুঝয়॥ | only the learned grasp that essence. |  |
| ভাব রস শব্দ অষ্টগণ লঘু গুরু। | Mood, rasa, word, the eight traits, light and heavy — | 301 |
| ছন্দ উক্তি শুদ্ধ হলে গাঁথনি সুচারু॥ | pure metre and phrase make the work fine. |  |
| এসব বিচারি লেখে সেই মহাকবি। | The great poet writes weighing all this; |  |
| নেহ নিজ মনোগত সেবে কেহ ভাবি॥ | not what one merely fancies in his mind. |  |
| কবি বাক্য রশ দহ অঙ্গ যুবা স্তন। | The poet's words, like a young breast, hold ten beauties — |  |
| আন ভাতি যে মোহয় রশিকের মন॥ | they enchant the connoisseur with rare splendour. |  |
| দাতা দান হেন্তে বাক্য হয় রস যুক্তা। | As is the giver's gift, so words gain savour; |  |
| শ্রোতিজল দানে ছিপি উদ্দারয় মুক্তা॥ | given the ear's water, the oyster yields its pearl. | 308 |
| যে বলে বলুক আমি দৈবে হীন মতি। | Let any say what they will — by fate my wit is weak. |  |
| নির্দোষী আছয় মাত্র ত্রিজগত পতি। | Only the Lord of three worlds is faultless. |  |
| সমুদ্রেত ডুব দিলে করি বহ যত্ন॥ | If one dives the ocean with great effort, |  |
| কেহ বট পায় কেহ বহুমূল্য রত্ন | some find a fig, some a precious jewel — |  |
| যদ্যপি | even so, |  |
| রত্নের আছে গ্রাহক বহল। | the jewel has many buyers, |  |
| সদত কার্যেত লাগে বট অল্প মূল॥ | yet the cheap fig still serves daily need. | 315 |
| উদ্যানের মেধ্য মিষ্ট ফল সুরুচির। | The garden's sweet, fine fruit — |  |
| অল্প প্রয়োজনে লাগে সতত জামির। | the lemon still serves small daily needs. |  |
| অগ্রগামি সকলে লুটিল পুণ্য ধন। | Those before plundered all merit's treasure; |  |
| বহ যত্রে অল্প পায় পৃষ্টগামী জন॥ | for all their toil, latecomers gain little. |  |
| এতেক ভাবিয়া মোরে ক্ষেমিবা সকলে। | So thinking, let all forgive me; |  |
| যে বলে বলয় যত্রি যন্ত্রে সেই বলে॥ | say what they will — the instrument speaks by its player. |  |
| যত কিছু নবীন যেতক পুরাতন। | Whatever is new and whatever old — | 322 |
| সকলের শ্রেষ্ঠ হৈল বচন রতন॥ | the jewel of speech is best of all. |  |
| জগজন নিযোটের যেতক জর্মিল। | Of all born in the world's close circle, |  |
| নবচন সমান অপত্য প্রসিবল॥ | none came forth equal to speech. |  |
| না কহিলে কথন সকল থাকে ধন্দ। | Unspoken, all things stay in confusion; |  |
| প্রকাশ করিলে সেবে বুঝে ভাল মন্দ॥ | once spoken, the mind knows good from bad. |  |
| অলখ নির্দোষ বাক্য যেন জীবর্তমা। | The unseen, faultless word is like the soul; |  |
| পুস্তক ভাঙারের রত্ন অতি নিরুপমা॥ | the book's treasure-jewel is past compare. | 329 |
| যেতক অশ্রুত মর্ম বচনে প্রকাশ। | What is unheard, speech brings to light. |  |
| প্রকটয় যেতক সকল ইতিহাস॥ | All history is shown through it. |  |
| ভারি দেখ যেতক সৃজিছে করতার। | See how much the Creator made weighty. |  |
| কেবল বচন বিনে কিবা আছে আর॥ | Yet without speech, what else is there? |  |
| পূর্বের রহস্য যত স্মরণ আছয়। | All past secrets held in memory — |  |
| বুঝহ বচন বিনে আর কেহ নয়॥ | know that without speech none can grasp them. |  |
| যদি অন্য রত্ন হৈত বচন সমতুল। | Were any jewel equal to speech — | 336 |
| বাক্যজালে শুক সারি পক্ষি বহ মূল॥ | in word's net even parrots and mynas hold worth. |  |
| বুদ্ধি বাক্য হেন্তে বহ সৃজিল ঈশ্বরে। | God made mind and speech in such plenty. |  |
| তাহার মহন্ত কেন কহিবারে পারে॥ | Who then can tell His greatness? |  |
| বুদ্ধিমেন্তে আপনারে হৈল হীন জ্ঞান। | By intellect itself, one's knowledge proves small. |  |
| এথা ওথা নিত্য নিত্য বাড়য় সম্মান॥ | Here and there, honour grows and grows. |  |
| জ্ঞান অনুরূপে জান আপনা মরম। | Know your own self by your knowledge. |  |
| তথাপি কিঞ্চিৎ লোক করয় ভ্রম॥ | Yet some still fall into some error. | 343 |
| কিঞ্চিৎধিক করিলে আপনা অপমান। | To think oneself even a little higher is self-disgrace. |  |
| জ্ঞানবন্ত মনে সেই নেহ বস্তু জ্ঞান॥ | To the wise, that is no true knowledge. |  |
| তাহার মূর্খতা নখয় অনু দিন। | His folly does not lessen day by day. |  |
| বুদ্ধিমেন্তে আপনাকে নভাবে প্রবীন॥ | By intellect he deems himself not yet wise. |  |
| তথাপিহ লোক সব কলির লক্ষণে। | Yet all, by the signs of the Kali age — |  |
| মূড় বুদ্ধি অমূল নবুজে কোন জনে॥ | dull-minded, none tells worthless from worthy. |  |
| কদাপি গোপত নেহ চতুরের চিতে। | From the clever heart nothing stays hidden. | 350 |
| কার পুঞ্জিধিক কে বন্ধ্য হীন॥ | Who holds more, and who is barren and low. |  |
| রিতে ধনবন্ত জেন সকলেরে ভাষে ধনি। | Likewise the rich call all others rich. |  |
| গুনিরে নিগুনি হেন ভাবয় নিগুনি॥ | The worthless deems the worthy worthless too. |  |
| যদ্যপি সকলে চাহে আপনার হিত। | Though all seek their own good — |  |
| সাধুজন মন পর কার্যেত বাঞ্ছিত॥ | the good man's heart wants others' good. |  |
| কেবল আপন হিত চিন্তে যেই জন। | Who thinks of his own gain alone — |  |
| পর অপকার কথা আছে তার মন॥ | in his heart lurks others' harm. | 357 |
| যার চিত্ত মন্দ ভাব মন্দ ফল পায়। | An evil nature reaps an evil end. |  |
| ভালেই এথা ওথা কুশলে গোঙায়॥ | The good fare well both here and there. |  |
| ধিরেবেক ধিরেই যদি ফুটে কাঁটা। | If a thorn pricks, bear it with patience. |  |
| হীন স্থানে কহিলে না লৈব দুঃখ বাটা॥ | Tell it to base folk, you get no pity. |  |
| দুঃখ পাই কান্দিলে হাসয় খল জন। | Weep in grief, and the wicked laugh. |  |
| নবুজে পামর লোকে দৈব নিযোজন॥ | The base do not grasp fate's decree. |  |
| তার পাশে দুঃখ না কহিও কদাচিত। | Never tell your grief to such a one. | 364 |
| চিন্তাকুল জন দেখি হয় হরিষত॥ | Seeing one sunk in care, he is glad. |  |
| ক্ষুধাতুর জন আগে না খাইও রুটি। | Eat no bread before a starving man. |  |
| যদি খাও সকলের দিবা কিছু বাঁটি॥ | If you eat, share with all there. |  |
| নির্ধনের আগে যদি সুবর্ণ তোলায়। | Weigh gold before a pauper — |  |
| দেখি অঙ্গ মোড়ায় ধনের সর্প প্রায়॥ | and he writhes like the serpent of wealth. |  |
| যদি বা বসন্ত বায়ু সুশীতল লাগে। | And if the cool spring breeze feels sweet — |  |
| উষ্ণমন্ত দীপ ন জালিবা তার আগে॥ | light no hot lamp before one who burns. | 371 |
| বুদ্ধি পাছে দামে সে উজ্জুল কলেবর। | Wit, after, gives that bright body worth. |  |
| তৃণ ভক্ষ হেতু প্রভু ন সৃজিছে নর॥ | The Lord made not man to graze on grass. |  |
| সে মনুষ্য ধিক জেন শীসের মাহাত্ত্য। | Shame on the man no greater than a grass-blade — |  |
| যেবে দৃষ্টি তৃণ পরে গর্দভের মত॥ | whose eye seeks grass like a donkey's. |  |
| সে সেই মনুষ্য যেই করে পর হিত। | He alone is a man who does others good. |  |
| কিবা দানে সন্তোষিব দুঃখিতের চিত॥ | or gladdens the afflicted heart by giving. |  |
| রুক্ষ দানে যত পুষ্প চন্দন সুগন্ধ। | Like flowers and sandal given as a harsh gift. | 378 |
| নিস্ফল মনুষ্য জন্ম হৈলে মতি মন্দ॥ | Man's birth turns vain when the mind turns base. |  |
| এমত শুনেছি মহা পণ্ডিতের মুখে। | So I have heard from great scholars. |  |
| দিব্য ভাবে শুতিলে স্বপন ভাল দেখে॥ | Who sleeps in a holy state sees fair dreams. |  |
| জন্ম কাল তাহার মরণ অতি ভাল। | Fair is the birth and death of such a one. |  |
| যার মন্দ চরিত্র অবধি মৃত্যুকাল॥ | Whose nature stays base till the hour of death. |  |
| সুপবিত্র চরিত্র যাহার জন্ম হয়। | One born with a wholly pure nature — |  |
| মরণ জীবন ধিক কীর্তি সংক্ষরয়॥ | Shame on life and death; only fame stays. | 385 |
| জীবন অবধি ফল মরণ সর্বথা। | Life's span is the fruit; death comes all ways. |  |
| যাহার মরণে লাগে লোক মনে ব্যথা॥ | He at whose death men's hearts feel pain. |  |
| কঠিনতা গর্ব তেজে যদি হয় ভাল | If one drops harshness and pride and turns good — |  |
| বিপত্তি হইলে তুমি শিরেবেক কাল। | In calamity, death waits at your head. |  |
| তাকে মহাজন মনে ভাবয় সদায়॥ | The great always count such a one great. |  |
| মৃত্তিকা গঠন অঙ্গ মৃত্তিকার প্রায়। | The body shaped from clay is kin to clay. |  |
| আন্ধার মৃত্তিকা মেধ্য উজ্জুল স্মৃতি॥ | The clay is dark and foul, yet memory shines. | 392 |
| পুষ্পেত গোলাব পুষ্প কণ্টক সঙ্গতি। | Among flowers, the rose keeps thorns for company. |  |
| কে বুঝিতে পারয় ঈশ্বর সুষ্কনু লীলা॥ | Who can grasp the Lord's subtle play? |  |
| সর্প হেন্তে উপজয় রিপু জীর্ণ শীলা। | From the snake springs the foe of crumbled stone. |  |
| বিষে বিষ জন্য মণি আছে সর্প সঙ্গে॥ | In its poison, by poison, the snake keeps a jewel. |  |
| কেবল কুমতি বিনা নাহি খল অঙ্গে। | Save ill-will, nothing fills the wicked's body. |  |

| আদ্যভাগে কহিছেন্তু মহন্ত নেজামি॥ | Great Nizami told this in the earlier part. |  |
| কহিতে সে সব কথা নাহি পারি আমি। | I cannot tell all those things. |  |
| অপেপ অধিক বুঝে পণ্ডিত গুণাধার॥ | The virtue-rich scholar grasps more on his own. |  |
| সমুখে পুস্তক কথা আছে মহা ভার। | Ahead the book's matter weighs heavy. | 413 |
| তেকারণে পরিত্যাগি বহল বচন॥ | So I leave off lengthy speech. |  |
| পুস্তকের সুর কেবা শুন ধীর জন॥ | Now hear, patient souls, the book's song. |  |
