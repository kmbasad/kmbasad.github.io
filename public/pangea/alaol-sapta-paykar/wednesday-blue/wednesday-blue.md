# বুধবার — নীল মণ্ডপ

## বুধবারের প্রসঙ্গ লাজ পরী কন্যার বিবরণ

| বাংলা | # |
| :-- | --: |
| *(রাগ আশাবরী জমক ছন্দ)* |  |
| বুধবারে বাহরাম নৃপ মহামতি। |  |
| কেবাদ ফিরোজ বর্ণ গৃহে কেলা গতি॥ |  |
| নানান সুগন্ধি পরী মন হরিষেতে। |  |
| মত কর্মে চলিলেক ফিরোজার ভিতে॥ |  |
| বরবস্ত্র হরচ্ছত্র ভূষিয়া সুছন্দে। |  |
| স্বর্গবৎ টাঙি মাঝে চলিল আনন্দে॥ |  |
| খোয়ারাজি রাজ কন্যা লাজ পরী নাম। |  |
| পরিয়া কেবাদ বর্ণ বস্ত্র অনুপাম॥ |  |
| ফিরোজার রথু অলঙ্কার সর্ব অঙ্গে। |  |
| গৃহের সীমায় আসি দাঁড়াইল রঙে॥ |  |
| গোমেদ ভূষিত অঙ্গে শত সংখ্যা সখি। |  |
| ক্ষেমা নাশি মৃদু হাঁসি সুধা চারুমুখি॥ |  |
| সৌরভ পূর্ণিত অঙ্গ গন্ধ পায়ে হাতে। |  |
| নক্ষত্র মণ্ডলে যেন শশধর সাতে॥ |  |
| হেনকালে বাহরাম তথা উপস্থিত। |  |
| রঙিম কটাক্ষ শরে হইল মোহিত॥ |  |
| মৃদু হাসি সুধা দৃষ্টি করি কলাবতি। |  |
| করেতে ধরিয়া সান্তাইল নিজ পতি॥ |  |
| গৃহের মাঝারে নিয়া করিল প্রবেশ। |  |
| নানা সুখরসে হৈল দিন অবশেষ॥ |  |
| রজনীতে নির্বাহিল কেলি-কলা রতি। |  |
| নৃপ বলে এক কথা কহ কলাবতি॥ |  |
| ভূমে শির দিয়া কন্যা করি আশীর্বাদ। |  |
| আয়ু রিদ্ধি বাঞ্ছা সিদ্ধি পুরেক মনসাধ॥ |  |
| নৃপ মন-বশ প্রায় নাহি জানি কথা। |  |
| না শোভে অসল জল সুধা ভক্ষ যথা॥ |  |
| তবে কি ঈশ্বর আজ্ঞা না যায় লংঘন। |  |
| প্রকাশিয়ু যেন মত আছেয়ে স্মরণ॥ |  |
| সাম দেশে পুরুষ মোহন নামে এক। |  |
| ধনবন্ত বিদ্যাবন্ত রূপে অতিরেক॥ |  |
| সর্ব দেশে প্রচার আছিল তার গুণ। |  |
| অতিথ ভক্তিয়ে ধর্মে কর্মেতে নিপুণ॥ |  |
| দিব্য এক উদ্যান পূর্ণিত ফুল ফল। |  |
| মাঝে মাঝে টাঙি সব অধিক উজ্জ্বল॥ |  |
| উজ্জ্বল যামিনী এক দিবস সমান। |  |
| মন সুখে গৃহেতে করিল সুরাপান॥ |  |
| প্রবিন জ্বালায় অঙ্গ ছটফট করে। |  |
| নিষ্ক্ষুরি বসিল গিয়া টিঙির উপরে॥ |  |
| মধু জিন টিঙিতে বহয় শুদ্ধ বাও। |  |
| শীতল মৌরভে শীঘ্র যুড়াইল গাও॥ |  |
| অন্দর বাহিরে বিস আছে একাষ্থর। |  |
| আসিয়া পুরুষ এক মিলল সত্বর॥ |  |
| নিকেট আইল যদি দেখিয়া চিনল। |  |
| ভাগি করি তাহারে বাণিজ্যে পাঠাই ছিল॥ |  |
| বলিলেক নিশি বহি গেলেক প্রহর। |  |
| হেনকালে কি লাগি আইলা একাষ্থর॥ |  |
| প্রণামিয়া কহিল বচন সবিশেষ। |  |
| দূরের গমনে হৈল দিন অবশেষ॥ |  |
| তৃণ বিনে দুঃখ পায় ক্লান্ত রথখর। |  |
| দেশে প্রবেশিতে নারি রহিল বাহির॥ |  |
| তোমারে দেখিতে অতি মন উত্তেরাল। |  |
| তেকারণে আসিনু না শুনি কার বোল॥ |  |
| তোমার দর্শনে মোর সাস্ত হৈল প্রাণ। |  |
| রহিতে না পারি পুনি যাইমু সেই স্থান॥ |  |
| বহু মূল্য দ্রব্য আনিয়াছি বহুতর। |  |
| হেম রতু আদি বস্ত্র কস্তুরি অম্বর॥ |  |
| সঙ্কট ত্বরিয়া আসি লংঘিলুম এথায়। |  |
| লভ্যাধিক একে দশ হইবে তথায়॥ |  |
| শুভ বার্তা কহিতে আইলুম এই স্থানে। |  |
| দেখিনু তোমারে এবে যাই তুষ্ট মনে॥ |  |
| এতেক কহিয়া ফিরি চলিল ত্বরিত। |  |
| মোহেনর মনে হৈল অতি হরষিত॥ |  |
| বহু ধন শুনি শ্রদ্ধা হৈল দেখিবার। |  |
| টঙ্গি হৈতে নামি শীঘ্রে মেলিল দুয়ার॥ |  |
| একাষ্থর চলিল তাহার পাছে পাছে। |  |
| পরিজন জানে সেই উদ্যানেতে আছে॥ |  |
| অগ্রগামী নিঃশব্দে চলিয়া যায় বেগে। |  |
| সেই গতি মোহেনে ধাইল লেগে লেগে॥ |  |
| দুই জাম অবধি ধাইল পাছে পাছে। |  |
| মোহেনে ভাবিল নিশি অল্প মাত্র আছে॥ |  |
| দুই দও বাট নেহ ধাই দুই জাম। |  |
| মদমত্ত ভ্রমে করিনু নষ্ট কাম॥ |  |
| ত্বরিতে চলিল বেগে মনে করি কোপ। |  |
| দেখিতে দেখিতে সঙ্গি হইল আলোপ॥ |  |
| চৌদিক পর্বত মাঝে ডাঙর প্রান্তর। |  |
| যাইতে পর্বত কাছে উদিল ভাস্কর॥ |  |
| প্রাতঃকালে চারিদিকে নিরক্ষিয়া চায়। |  |
| কোথা হৈতে কোথা আইল ভাবিয়া না পায়॥ |  |
| মত্ত ভোর শেষে জাগরণ চতুর জাম। |  |
| দূর স্থান ধাই আসি না হৈল বিশ্রাম॥ |  |
| অলক্ষিতে শরীর লাড়িতে নারে মেলি। |  |
| আঁখি প্রকাশিতে নারে পড়ে ঢলি ঢলি॥ |  |
| বহুল যতনে গিরি নিকেট আসিয়া। |  |
| মন দুঃখে রুক্ষতলে রহিল শুইয়া॥ |  |
| দুই জাম বহি যদি অরুণ হানিল। |  |
| রৌদ্র জালে ছান হৈয়া জাগিয়া উঠিল॥ |  |
| বন রুক্ষ প্রান্তর সঙ্গত কেহ নাই। |  |
| আক্ষেপ করিয়া কান্দে কি হৈল গোঁসাই॥ |  |
| *(রাগ ত্রিপদী ছন্দ)* |  |
| মেদ মত্ত হৈয়া ভোর, না বুঝিনু কার্য তোর, |  |
| ধন লোভে ছান হৈল মতি। |  |
| কিবা হৈল দৃষ্টি বন্ধ, না চিনিনু হই অন্ধ, |  |
| কর্ম দোষে এমন দুর্গতি॥ |  |
| কোথা হৈতে আইনু কোথা, পুনরপি যাইমু কোথা, |  |
| না পাইনু পন্থের উদ্দেশ। |  |
| কি মোর অশুভ দশা, না পুরিল মন আশা, |  |
| দুঃখ বশ হৈলুম অবশেষ॥ |  |
| নিশাকালে গতি বেগে, চরণ না চলে আগে, |  |
| ক্ষুধা তৃষ্ণা মরি রৌদ্র জালে। |  |
| হারাইনু নিজ বুদ্ধি, কেহ দিতে নাহি শুদ্ধি, |  |
| কোন হেতু তরিমু জঙ্গালে॥ |  |
| নয়নে বহয় নীর, এক বুদ্ধি নেহ স্থির, |  |
| ভাবি চিন্তি মনে হৈলুম সার। |  |
| কি ফল রহন এথা, চলি যাও যথা তথা, |  |
| দুঃখ সহি অঙ্গে আপনার॥ |  |
| ধীরে ধীরে তথা হৈতে, চলিল বিকট পন্থে, |  |
| বিশ্রাম করিয়া স্থানে স্থান। |  |
| না পারে চলিতে ধাপে, ক্ষুধায় শরীর কাঁপে, |  |
| তাতে হৈল দিন অবসান॥ |  |
| রজনী প্রবেশ কালে, বসিলেক তরুতলে, |  |
| চলিতে না পারে পদ গতি। |  |
| ছৈয়দ মহাম্মদ খাঁন, রস ধীর পুণ্যবান, |  |
| আলাওল মধুর ভারতি॥ |  |

## মোহেনর দানব সঙ্গে কেথোপকথন

| বাংলা | # |
| :-- | --: |
| *(জমক ছন্দ)* |  |
| হেনকালে রমণী পুরুষ দুইজন। |  |
| আচম্বিতে আসি তথা দিল দর্শন॥ |  |
| দোহানের কান্ধে ভর গমন ত্বরিত। |  |
| মোহেনেরে দেখিয়া হইল সচকিত॥ |  |
| রমণী অন্তরে রাখি নিকেট আসিয়া। |  |
| কহিল পিরীতি ভাবে বচন গাঞিয়া॥ |  |
| বলিল মনুষ্য তুমি ভব্য সুচরিত। |  |
| কি লাগি আসিছ হেথা ভীতান্ত ভূমিত॥ |  |
| এই স্থানে ভূত প্রেত যক্ষ বহুতর। |  |
| না আইসে কুঞ্জর ব্যাঘ্র মনে ভাবি ডর॥ |  |
| বুদ্ধিমন্ত হৈয়া কেনে আইলা এই ঠামে। |  |
| না আইসে নির্বোধ পশু প্রেতের আশ্রমে॥ |  |
| শুনিয়া মোহেনে তারে কহিল তখন। |  |
| নিজ ইচ্ছা না আসিছি শুন মহাজন॥ |  |
| যেমতে আইল তথা জানাইল আগে। |  |
| কহিল তোমার দেখা পাই মহা ভাগে॥ |  |
| রহিছি দুর্গম ভূমে পন্থ হারাইয়া। |  |
| বহু পুণ্য পাইবা যদি দেও উদ্দেশিয়া॥ |  |
| মোহন্ত পুরুষ তুমি বুঝ ধর্মাধর্ম। |  |
| জীব রক্ষা আর ধিক কি আছে সুকর্ম॥ |  |
| সে পুরুষে বলিলেক শুনহ উত্তর। |  |
| ভ্রমাই দানবে আনে শত শত নর॥ |  |
| ভাগ্যের প্রভাবে তোর রহিছে জীবন। |  |
| আজি নিশি রক্ষাকারী আমি দুইজন॥ |  |
| আমি দোহানের পাছে পাছে আইস চলি। |  |
| একাষ্থর কদাপি না রহ এই স্থলি॥ |  |
| এত শুনি মোহেনে চলিল পাছে পাছে। |  |
| ধীরগামী না লংঘয় শীঘ্রগামী কাছে॥ |  |
| শক্তি করি কতক্ষণ করিল গমন। |  |
| অবশেষে দুইজন হৈল অদর্শন॥ |  |
| চলিতে না পারে ধূপে ক্লান্ত ক্লান্ত মনে। |  |
| একাষ্থর পড়িয়া রহিলা সেই স্থানে॥ |  |
| রুক্ষ ডাল পত্র যত সুকোমল পায়। |  |
| না পারি রহিতে ক্ষুধা ছিড়ি ছিড়ি খায়॥ |  |
| সেই রাত্রি প্রভাতেত তৃতীয় প্রহর |  |
| পর্বতে পর্বতে ভ্রমিলেক একাষ্থর॥ |  |
| ক্লান্ত হৈয়া এক স্থানে পড়িয়া রহিল। |  |
| কতক্ষণে অথ পদ শব্দ যে শুনিল॥ |  |
| চক্ষু মেলি দেখে এক দিব্য অশ্ববার। |  |
| তাহা দেখি লুকাইল শিলার মাঝার॥ |  |
| অর্ধ অঙ্গ গুপ্ত অর্ধ রহিল বাহিরে। |  |
| অপ্রবেশে সর্ব অঙ্গ সেই গর্তান্তরে॥ |  |
| শব্দ পাই অশ্ববারে চাহিল ফিরিয়া। |  |
| তর্জিয়া পুছিল তারে নিকেট আসিয়া॥ |  |
| কোন হেতু এথা আইলে কহ বার্তা সার। |  |
| এই স্থল যোগ্য নেহ আসিতে তোমার॥ |  |
| যদি সত্য না কহ করিমু দুই খান। |  |
