# শুক্রবার — শুভ্র মণ্ডপ

## শুক্রবারের প্রসঙ্গ রোজ-আপজুনী কন্যার বিবরণ

| বাংলা | # |
| :-- | --: |
| *(জমক ছন্দ)* |  |
| রজনী প্রভাতে বাহরাম নরপতি। |  |
| শুক্রে অধিষ্ঠান গৃহে যাইতে কৈল মতি॥ |  |
| শুক্রবর্ণ গৃহেত চলিল মহারাজ। |  |
| শুক্র বস্ত্র শুক্র ছত্র আদি নানা সাজ॥ |  |
| সেই গৃহে মগরিব নৃপতি দুহিতা। |  |
| রোজ-আপজুনী নাম অতি সুচরিতা॥ |  |
| শ্বেতবর্ণ ভূষণ হিরার অলঙ্কার। |  |
| শ্বেত পুষ্পমালা অঙ্গে সুছন্দ সুনার॥ |  |
| সখিকুল শ্বেতবাস পুষ্প অলঙ্কার। |  |
| হংসরাজ ঝাঁক যেন ক্ষীরোদ মাঝার॥ |  |
| চন্দন আবির হৈতে কস্তুরির ধুলি। |  |
| করে লৈয়া আঙু হৈয়া সব চন্দ্রবালি॥ |  |
| নৃপতি সহিতে যদি হৈল সমদৃষ্টি। |  |
| কটাক্ষেতে করিল সৌরভ পুষ্পবৃষ্টি॥ |  |
| সে কটাক্ষ ঘারে নৃপ হইতে মোহিত। |  |
| করেত ধরিলা বালা হাসিয়া ইঙ্গিত॥ |  |
| নৃত্য গীতে উল্লাসীতে গৃহে প্রবেশিল। |  |
| সমস্ত দিবস ভোগ রসে নির্বাহিল॥ |  |
| নিশা আদ্যে আনন্দিত হৈয়া নরপতি। |  |
| কহিল প্রসঙ্গ এক কহ গুণবতী॥ |  |
| প্রণামিয়া আশীস পূর্বকে বরবালা। |  |
| কহিল উত্তম কথা কি জানি অবলা॥ |  |
| তবে কি ঈশ্বর আজ্ঞা না যায় লঙ্ঘন। |  |
| তেকারণে মনোগত প্রকাশ বচন॥ |  |
| এক বৃদ্ধ রমণী কহিতে মাতৃ স্থানে। |  |
| সেই কথা প্রবেশ করিছে মোর কাণে॥ |  |
| কস্তিনা দেশে ছিল সাধু গুণধাম। |  |
| তনয় উত্তম তার হুমাউন নাম॥ |  |
| পরম সুন্দর তনু কামদেব জিনি। |  |
| অস্ত্রে শাস্ত্রে বিদ্যায় পারগ বহু গুনি॥ |  |
| অধিক আশ্চর্য ধন দাতা ক্ষমাশীল। |  |
| রূপে গুণে তার সম সে কালে না ছিল॥ |  |
| কোটি ধন লাগাইয়া কিনিল উদ্যান। |  |
| ইন্দ্রের নন্দন বন না হয় সমান॥ |  |
| পবিত্র পাষান গঠ হেম রতু লয়। |  |
| কেয়ারী ঝরনা হেরি দেব মুনি মগ্ন॥ |  |
| ইচ্ছা হৈলে আইসে সেই উদ্যান মাঝার। |  |
| ক্ষণে একাক্ষর ক্ষণে সঙ্গে পরিবার॥ |  |
| এক নিশি গৃহেত বসিয়া হুমাউনে। |  |
| মনোহর যন্ত্র শব্দ শুনি উপবনে॥ |  |
| পক্ষখুরে মিলি মধুষের সুললিত। |  |
| শুনি ধরাইতে নারে বিজ্ঞকুল চিত॥ |  |
| শুনিয়া কুমার মন ছট ফট করে। |  |
| গৃহ হৈতে আইল সেই উদ্যান দুয়ারে॥ |  |
| প্রবেশিতে না পারে দুয়ার দেখে বন্ধ। |  |
| চতুর্দিকে পন্থ না পাইয়া হৈল ধন্দ॥ |  |
| মনে ভাবে নেহ এই মনুষ্যের কর্ম। |  |
| কিবা দেব অপ্সরা বুঝি যুক্ত মর্ম॥ |  |
| না জানি কেমন হৈতে হেন যন্ত্র বাহে। |  |
| সাফল্য জীবন তার এ রঙ যে চাহে॥ |  |
| মনে দড়াইল তার শঙ্কা পরিহরি। |  |
| শতাব্দ জীবন ধিক তিলে দেখি মরি॥ |  |
| জল নিষরণ পন্থ আছে চারি ভিতে। |  |
| এক রন্ধ্র পাশে গেল অস্ত্র করি হাতে॥ |  |
| সেই স্থানে করি নিজ অঙ্গ যুক্ত বাট। |  |
| প্রবেশিয়া দেখিল উজ্জ্বল চন্দ্র হাট॥ |  |
| নানা নৃত্য করয় সুন্দরী অপ্সরী। |  |
| উদ্যানের ফুল সব ভরিয়া কবরি॥ |  |
| সেই নৃত্যকির রূপ দেখিতে আচম্বা। |  |
| দেখিতে নিছনী যায় তিলোত্তমা রম্ভা॥ |  |
| ধন্দ হৈয়া চাহে সেই উগ্র দুই অাঁখি। |  |
| চোর বলি প্রহরী ধরিল তারে দেখি॥ |  |
| দেশ পাঁচে ধরিয়া বাঁধিল হৈতে পেল। |  |
| বলিল দারুণ চোরা কোথা হৈতে আইলে॥ |  |
| কুমারে বলিল তবে এথা নাহি চোর। |  |
| উদ্যান ঈশ্বর আমি এ উদ্যান মোর॥ |  |
| সুললিত যন্ত্রকুল গৃহ হৈতে শুনি। |  |
| নির্ণয় করিতে আইলুং কার যন্ত্র ধ্বনি॥ |  |
| দ্বার না পাইয়া রন্ধ্র পেষ্ঠে প্রবেশিলুং। |  |
| রূপে ভঙ্গে নৃত্য গীতে ভুলিয়া রহিলুং॥ |  |
| মন বন্দী হৈল মোর দেখি তোমা সব। |  |
| কি লাগিয়া হস্ত বাঁধি করহ লাঘব॥ |  |
| সে সবে শুনিয়া বলে শুনহ কুমার। |  |
| কি মেতে প্রত্যয় করি উদ্যান তোমার॥ |  |
| উদ্যানের চিহ্নু সব কহ বিরচিয়া। |  |
| তবে ছাড়ি দিব ঈশ্বরেরে দর্শাইয়া॥ |  |
| উদ্যানের চিহ্নু সব প্রকাশি কহিল। |  |
| কুমারের রূপে সব যুবতী মোহিল॥ |  |
| কহিল এমন রূপ কভু নাহি দেখি। |  |
| ইহার দর্শনে ঠাকুরাণী হৈব সুখি॥ |  |
| এত শুনি বন্ধন খসাই সহসাত। |  |
| আদর করিয়া নিল কুমারী সাক্ষাৎ॥ |  |
| কুমারীকে কহিল পাইছি এক চোর। |  |
| স্বগর্বে পুরুষে বলে উপবন মোর॥ |  |
| তোমার সাক্ষাতে এই দর্শাইল আনি। |  |
| যুক্তি বিমষিয়া আজ্ঞা কর ঠাকুরাণি॥ |  |
| আপাদ মস্তক নিরক্ষিয়া ভাল মেতাঁ। |  |
| প্রেমানলে জ্বলি দেহে রহিল মুছিতে॥ |  |
| নয়নে দোহ চাহিয়া রহিল। |  |
| কতক্ষণে ধৈর্য্য ধরি চৈতন্য লভিল॥ |  |
| মনে মিলি গেল নয়নে। |  |
| অাঁখি পেষ্ঠে প্রবেশিল দোহান পরাণে॥ |  |
| তবে কন্যা পাট হৈতে সাদরে উঠিয়া। |  |
| বসাইল দক্ষিণ পার্শ্বে কুমারে তুলিয়া॥ |  |
| জিজ্ঞাসিল কুমার কুমারী স্থানে তবে। |  |
| আপনা রহস্য কথা মোরে কহ এবে॥ |  |
| কন্যা বলে গন্ধর্ব নৃপতি সুতা আমি। |  |
| পিতা স্বর্গ গেতে রাজ্য করি বিনু স্বামী॥ |  |
| যথাতে উত্তম স্থল আছে স্থানে। |  |
| বিহারীতে আসি হেথা আকাশ গমনে॥ |  |
| মনুষ্যের শক্তি আমা দেখিতে না পারে। |  |
| সেই দর্শন পায় দেখা দেই যারে॥ |  |
| বাপে বিভা না দিল না পাই যোগ্য জন। |  |
| একাক্ষরী রাজ্য পালি বাঁধি নিজ মন॥ |  |
| প্রতি দেশে সখিগণ সতত ভ্রময়। |  |
| সুপুরুষ কুলোত্তম কোথা না ঘটয়॥ |  |
| রূপে গুণে কুলোত্তম তোমারে দেখিয়া। |  |
| মোর স্থানে সখিগণে কহিলেক গিয়া॥ |  |
| না দেখিলে নিজ অাঁখে নাহিক প্রত্যয়। |  |
| তেকারণে এথা আইনু করিতে নির্ণয়॥ |  |
| দর্শন পাইয়া পুরিল মন সাধ। |  |
| আজুকা খণ্ড কণ্ঠ চক্ষের বিবাদ॥ |  |
| যেমত আমার মন দেখিয়া মিজিল। |  |
| তেমত তোমার চিত্তে প্রত্যক্ষে বাজিল॥ |  |
| যোগ্যজন দেখি মনে আমিহ সম্মতি। |  |
| মনে না ভাবিও মোরে চপলিত মতি॥ |  |
| পিতৃ শিরে নাহি একাক্ষরী ভুঞ্জি রাজ। |  |
| আপনা উদ্যোগ বিনু সিদ্ধ নেহ কাজ॥ |  |
| দেব আরাধিয়া আমা দেখিতে না পারে। |  |
| মুখ্য সখী বাক্য শুনি আইনু এথাকারে॥ |  |
| ব্যবহারে আহারে উচিত নেহ লাজ। |  |
| তেকারণে লজ্জা ত্যাজি কহি নিজ কাজ॥ |  |
| এবে কহ তোমার মনেতে কিবা আছে। |  |
| যে থাকে মরম বুঝি নিবেদিব পাছে॥ |  |
| শুনিয়া কুমার মন পূর্ণিত উল্লাসে। |  |
| এক লেফে হস্ত যেন লাগিল আকাশে॥ |  |
| কুমারে বলিল মোরে পরশন বিধি। |  |
| তেকারণে ভিনি যেতু মিলাইল মিধি॥ |  |
| অতি তপস্যার ফলে হেন কর্ম ঘেট। |  |
| আমি ক্ষুদ্র কি যোগ্যতা বসিতে নিকেট॥ |  |
| মোহন্তের বাক্য মাত্র করি সপ্রত্যয়। |  |
| এহেন অসম্ভ কর্ম কার মনে লয়॥ |  |
| কি লাগি জিজ্ঞাস মোরে বচন অসম্ভ। |  |
| যে কহিলা সে না হৈলে মৃত্যু মোর লক্ষ্য॥ |  |
| বিধি বেশে আসি মুই শরণ লইলুং। |  |
| চরণ কমল তলে মন সমর্পিলুং॥ |  |
| শুনি কন্যাবর অতি আনন্দিত হৈয়া। |  |
| পুনরপি কহিল কুমারে সন্ধোধিয়া॥ |  |
| বিকৃত মূরতি এক যক্ষ মহা কায়। |  |
| মোর ভাবে তার চিত্ত বিকল সদায়॥ |  |
| একবারে মোর পাশে আইসে নিত্য নিত্য। |  |
| প্রলাপ করিয়া ভাঙিও রাখি তার চিত্ত॥ |  |
| অপ্সরা হৈতে নাহি তাহার মরণ। |  |
| মোর হৈতে প্রাণে বাঁচিয়াছে তেকারণ॥ |  |
| সেই যক্ষ এথা যদি দেখে আন জন। |  |
| মারিবার নিমিতে করিব প্রাণপণ॥ |  |
| মহা গুণবন্ত তুমি জান নানা সন্ধি। |  |
| তোমার মন্ত্রণা যোগে হৈয়া যাইব বন্দি॥ |  |
| কপট স্বভাবে সব ভেদ লৈব আমি। |  |
| তন্ত্রে মন্ত্রে ছলে বলে সংহারিব তুমি॥ |  |
| আর নানা মন্ত্রণা করিয়া হরিষেতে। |  |
| ভক্ষ দ্রব্য আনিবারে বলিল ইঙ্গিতে॥ |  |
| কন্যা বলে কুমারেরে শুনহ বচন। |  |
| উচিত না হয় আজি একত্রে ভোজন॥ |  |
| তুমি ক্ষমাশীল ধীর আমি কন্যা সতী। |  |
| অল্পব্যাজে অনুচিত চপলিত মতি॥ |  |
| অসন্তোষ হও বলি মনে ভাবি ডর। |  |
| বিমর্শিয়া নিজ মনে চাহ গুণাকর॥ |  |
| কুমার বলিল বালা কন্যা কুলবতী। |  |
