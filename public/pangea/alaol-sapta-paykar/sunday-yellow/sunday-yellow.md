# রবিবার — পীত মণ্ডপ

## রবিবারের প্রসঙ্গ এরাকি নৃপের বিবরণ আদি

| বাংলা | # |
| :-- | --: |
| *(দোপদী ছন্দ রাগ)* |  |
| নিজ বারে প্রভাতে প্রচণ্ড দিবাকর। |  |
| উরি ভঙ্গ করিলা সৈসন্য সহোদর॥ |  |
| রবি অধিষ্ঠান টঙ্গি পিত বর্ণ ধরে। |  |
| পীত বাস পরিয়া চলিলা সেই ঘরে। |  |
| জর্কিস বাদলা আর দামাস্কে খোতিন। |  |
| পরিয়া চলিল বাহরাম গুণমনি॥ |  |
| শিরেতে সুবর্ণ তাজ উজ্জ্বল কিরণ। |  |
| বিধর্ম গ্রাসিল সুরে অপূর্ব কথন॥ |  |
| সেই গৃহবাসী রূমি নৃপতি নন্দিনী। |  |
| রূপে গুণে অলঙ্কৃত নামে হুমায়ুনি॥ |  |
| শত সংখ্যা সখি সঙ্গে পরি পীত বাশ। |  |
| তারক মণ্ডলে যেন রহিণী প্রকাশ॥ |  |
| পূর্ণ রতু মাঝে অলঙ্কত কুটা মণ। |  |
| ঝলকে অরুণ কিবা প্রভা সৌদামিনী॥ |  |
| সুললিত নৃত্য গীত কিবা শব্দ চালি। |  |
| নানা রঙ্গে অঙ্গে ভঙ্গে নাচে নৃত্যকালি॥ |  |
| নৃপতি আসিব ভাবে হৈয়া ষষ্টমানা। |  |
| গৃহের সীমায় আসি হৈল আঙ্গুরান॥ |  |
| হেনকালে বাহরাম তথা উপিষ্থত। |  |
| হেরি পতি সতী মতি উল্লাসিত চিত॥ |  |
| আড় আঁখি বক্র দৃষ্টি ভুরু মোড়াইয়া। |  |
| দাণাইল বাম পার্শে ভূমে চুষ দিয়া॥ |  |
| সেই লক্ষে ছিদ্র পাই বিজয় অতর্হ। |  |
| যুড়িল কটাক্ষ বাণ ধরি ভাও ধরু॥ |  |
| অনেক অব্যর্থ বাণে ভেদিল মরম। |  |
| রতি ভাবে হৈল নৃপ আপনা ভরম॥ |  |
| আত্মা নাশ মিত্র সার দাড়াইল মুক্তি। |  |
| যোগী ভোগী এই ভাবে তরনের উক্তি॥ |  |
| মোহিত হইতে নৃপ কলা বিজ্ঞ বালা। |  |
| নৃপ পাশে আইল যেন চমকে চপলা॥ |  |
| দরিদ্রের হেতে যেন বহু মূল্য নিধি। |  |
| প্রসন্ন হইয়া শীঘ্রে মিলাইল বিধি॥ |  |
| গেলে ধরি গাঢ় আলিঙ্গিয়া নরনাথে। |  |
| আঁখি মুখে চুষিয়া ধরিল শীত্র হাতে॥ |  |
| উৎসব আনন্দে রাজা গৃহে প্রবেশিল। |  |
| কন্যা সঙ্গে কেলি রঙ্গে পাটেত বসিল॥ |  |
| সুখে ভোগে কেলি রসে দিন অবশেষা। |  |
| প্রকাশিল তারাগণ আলুপ দীবেশ॥ |  |
| সম্পূর্ণ ভোজন রতি শেষে নরপতি। |  |
| শয়ন সমর আদেশিলা কন্যা প্রতি॥ |  |
| কহ গুণবতি এক দিব্য উপকথা। |  |
| উপভক্ষাও আনন্দ খণ্ডাও মন বেথা॥ |  |
| রাজার আরতি শুনি রূম রাজবালা। |  |
| প্রকাশিল বচন রসদ চারুকলা॥ |  |
| চিরজীবি হও রাজা শত্রু হোক নাশ। |  |
| রহোক অখণ্ড যশ জগতে প্রকাশ॥ |  |
| যেই শিরে না করে তোমারে দণ্ডবৎ। |  |
| সগর্ভে তাহার হোক শীঘ্র মুণ্ড পাত॥ |  |
| মোহস্ত শ্রবণ যোগ্য না জানি কখন। |  |
| ঈশ্বর বচন নারি করিতে লংঘন॥ |  |
| তেকারণে নিবেদিমু কথা অনুমত। |  |
| ঈশ্বর আজ্ঞার যুক্ত হৈয়ে অনুগত॥ |  |
| *(রাগ দোপদী ছন্দ)* |  |
| এরাক দেশেতে ছিল এক মহীপাল। |  |
| তার আজ্ঞাপাল ছিল সব পূর্বকাল॥ |  |
| অতুল সম্পদ সৈন্য ছিল নৃপবর। |  |
| রাজাগণে আসিয়া দ্বারেত দিত কর॥ |  |
| নানা বিদ্যা পারগ বহুল সুখ ভোগ। |  |
| পাটে মাত্র মহাদেবী না ছিল সংযোগ॥ |  |
| কহিল জ্যোতিষে গণি রাশি গ্রহ ভাবি। |  |
| পাটেখরী বিহনে রহিব চিরজীবি॥ |  |
| মুখ্য মহাদেবী যদি থাকে রাজপাটে। |  |
| ছত্র ভঙ্গ অপায়ু তুরিত আসি ঘটে॥ |  |
| আপিন হইয়া বিজ্ঞ চাহিল গণিয়া। |  |
| বিনি দেবী রাজ্য পালে এ মর্ম জানিয়া॥ |  |
| কতকাল এমেতে বহিল একাখর। |  |
| সধর্ম যৌবন হৈল মদন প্রখর॥ |  |
| অতি তীক্ষ্ণ কাম শর সহিতে না পারি। |  |
| কিনয় সহস্র সংখ্যা পরম সুন্দরী॥ |  |
| নিশি দিশি রাজ সেবা করয় কর্কশ। |  |
| দাসীবৎ থাকিয়া ভুঞ্জয় সুখ রস॥ |  |
| রূপ রস সেবা বশ হইল নৃপতি। |  |
| দৈবগতি আপনারে হইয়া বিষয়তি॥ |  |
| পাটেখরী ভাব ধরি মেন করি গর্ব। |  |
| দিনে দিনে সেবা হীনে নাশে দর্প সর্ব॥ |  |
| ক্রোধে অতি নরপতি অগ্নিসম হয়। |  |
| অতি দুঃখে প্রাণী রাখে নারী বধ ভয়॥ |  |
| তবে তারে আপনার গৃহে না রাখিয়া। |  |
| যথা তথা বেচএ উচিত মূল্য লৈয়া॥ |  |
| এই মেতে বহু দাসী বেচিল নৃপতি। |  |
| কিংকর বিকৃত হৈল রাজার অখ্যাতি॥ |  |
| যারে কৃপা করে সেই হয় গর্ভধারী। |  |
| পুনি বেচে মন দুঃখ সহিতে না পারি॥ |  |
| বারে বারে একাখর পাটেত গোঁয়ায়। |  |
| মেনের মানস যোগ্য রমণী না পায়॥ |  |
| সেবা হেতু আছয় যেতেক নারীগণ। |  |
| চলাচল হয় বিনু মুখ্য একজন॥ |  |
| চিতারতি অনুরূপ না পাইয়া ভাল। |  |
| একাখর পাটেত বঞ্চয় কত কাল॥ |  |
| একদিন মনুষ্য বিক্রেতা বণিজ্যার। |  |
| চীন দেশ হেতে আইল এরাক মাঝার॥ |  |
| শত সংখ্যা সুন্দরী আনিল মেনারমা। |  |
| তাহে এক কন্যা যেন তারক চন্দ্রিমা॥ |  |
| মৃগমন্দ সৌরভ সঘন কেশ-ভার। |  |
| নয়ান পুষ্টিকা নব মরক্ত আকার॥ |  |
| কন্দর্প কুণ্ডল ভুরু নীলোৎপল আঁখি। |  |
| কটাক্ষে ভুবন মোহে কর্ণ গৃধ পাখি॥ |  |
| খগ চক্ষু জিত নাসা বান্দুলি অধর। |  |
| সুপাকা দাড়িষ বীজ দর্শন সুন্দর॥ |  |
| কনক মুকুর মুখ কমল নিন্দিত। |  |
| গিম নীলকণ্ঠ কুন্তু দেখি বিরাজিত॥ |  |
| ফষ বিল্যান্তুল কটি জিনিয়া ডুষকু। |  |
| নিকর্কশ কমল মৃণাল ভুরু চারু॥ |  |
| রক্তপদ্ম কর হেম চম্পক অঙ্গুলি। |  |
| পলটি কদলী রামা উরুযুগ বলি॥ |  |
| চরণ কমল পদ্ম জগমন লোভা। |  |
| কুশল দ্বাদশ বাণ জিনি অঙ্গ প্রভা॥ |  |
| ব্রহ্মা ইন্দ্র বাহন জিনিয়া গতি লীলা। |  |
| সুবাসিত ঈষিত ভাসিত সুধা নিলা॥ |  |
| নব রঙ্গে চারু ভঙ্গে হেরি ঘন মোহে। |  |
| রত্না রুচি রতি সৃষ্টি জিনি রূপ চোহে॥ |  |
| নৃপতির স্থানে আসি কহিলেক চরে। |  |
| আনিতে আদেশ হৈকল আমন্দ বিভোরে॥ |  |
| শত সংখ্যা দোল্লাছুলি সঙ্গে সদাগর। |  |
| আনিল নৃপতি স্থানে আনন্দে বিস্তর॥ |  |
| একে একে নৃপতি দেখিল সর্ব জনা। |  |
| নবীন বয়সী সব রূপে সুলক্ষণা॥ |  |
| একত্র করিয়া পুনি দেখিল স্বরূপ। |  |
| সুচারু সুঠাম সব আপনার রূপ॥ |  |
| রূপবতী সমান না হয় এক বালা। |  |
| সূর্য্য দৃষ্টে না শোভে শেতক চন্দ্রকলা॥ |  |
| মধ্যে দাণাইল নৃপ রূপবতী সঙ্গে। |  |
| বেষ্টিত যুবতী কুল যুড়ি অঙ্গে॥ |  |
| যদ্যপি সকল কন্যা রূপেত আগলি। |  |
| নিশাকর মধ্যে যেন নক্ষত্র মণ্ডলি॥ |  |
| সুরূপ মণ্ডলি প্রতিষ্ঠীত মনুহরা। |  |
| সুবর্ণের যরে যেন চন্দ্রের কাঙ্কুরা॥ |  |
| দেখি দেখি বিভোর হইয়া নরপতি। |  |
| ইচ্ছিল বহুল ধনে লৈতে রূপবতী॥ |  |
| নৃপতি কহিল যত চাহ দিব ধন। |  |
| কহু শুনি এ রমণী চরিত্র কেমন॥ |  |
| ভূমি চুম্পি বণিজ্যার কহিল তখন। |  |
| আসিয়া ভিক্ষিলুং এথা নৃপতি লবন॥ |  |
| যদি ধন লই মুক্তি করিয়া প্রলাপ। |  |
| দুই মত দোষ মানহানি আর পাপ॥ |  |
| সত্য ছাড়ি অন্য না কহিব নৃপ আগে। |  |
| অসত্য বাদীর অঙ্গ বেতক না লাগে॥ |  |
| যেন মত রূপবালা সেবায় তৎপর। |  |
| এক দোষ মাত্র তার আছে গুরুতর॥ |  |
| রতি ক্রিয়া সম্মত না হয় কদাচন। |  |
| অতি গ্রহ কলে চাহে তেজিতে জীবন॥ |  |
| যেই জনে বহু ধনে লয় গুণবতী। |  |
| ক্রোধ করি দেয় ফিরি রাখি এক রাতি॥ |  |
| তাহারে কিনিলে নৃপ দুঃখ পাইবা মেন। |  |
| অন্য জনে লও কিনি দিমু বিনা ধনে॥ |  |
| শুনি উপজিল দুঃখ নৃপতির চিত্তে। |  |
| অতি রূপ দেখি চিত নারে ধরাইতে॥ |  |
| মেন ভাবে নবিন বয়সি হিন রস। |  |
| এক রাতে কেমতে হইব মন বস॥ |  |
| অপে অপে প্রেম ফান্দে বাজাইলে মন। |  |
| রতি শ্রদ্ধা জন্মাইব প্রবল মদন॥ |  |
| এই মেতে নরপতি মেনত ভাবিয়া। |  |
| সেই রূপবতী লৈল বহু রতু দিয়া॥ |  |
| আর বহু প্রসাদে তুষিলা সদাগর। |  |
| কন্যা লৈয়া গৃহে প্রবেশিল নৃপবর॥ |  |
| দিব্য অলঙ্কার বস্ত্র দিয়া নরনাথ। |  |
| ধনের কুলুপ কুঞ্জি সুপিলেক হাত॥ |  |
| প্রেমবাক্যে আশ্বাস করিয়া বারে বার। |  |
| মোর রাজ্যধন প্রাণ সকল তোমার॥ |  |
| সেবে রাত্রে সেবা মোরে করিবা যতনে। |  |
| পাটেখরি ভাব মাত্র নরাখিবা মেন॥ |  |
| কন্যা বলে দাসী আমি সেবার আরতি। |  |
