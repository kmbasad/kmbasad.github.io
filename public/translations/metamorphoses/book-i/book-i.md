| আমার মন চাইছে বলতে নতুন রূপে রূপান্তরিত | In nova fert animus mutatas dicere formas | 1 |
| বস্তুর কথা; রূপ বদলের রূপকার হে সব দেবতা, | corpora; di, coeptis (nam vos mutastis et illas) |  |
| আমার মহাকাব্য অনুপ্রাণিত করে অবাধে বহাও | adspirate meis primaque ab origine mundi |  |
| মহাবিশ্বের শুরু থেকে আমাদের বর্তমান অবধি। | ad mea perpetuum deducite tempora carmen. |  |
| | | |
| সমুদ্র মাটি আর সব-ঘেরা আকাশের জন্মের আগে | Ante mare et terras et quod tegit omnia caelum | 5 |
| দুনিয়ার সবখানে প্রকৃতি দেখাত কেবল এক রূপ, | unus erat toto naturae vultus in orbe, |  |
| যার নাম কেয়স: খাপছাড়া জিনিসের রুক্ষ পিণ্ড, | quem dixere chaos: rudis indigestaque moles |  |
| এক গাদা অচেতন পদার্থ যার বেমানান মৌলের | nec quicquam nisi pondus iners congestaque eodem |  |
| পরস্পরবিরোধী সব বীজ এক জঞ্জালে একাকার। | non bene iunctarum discordia semina rerum. |  |
| তখনো পৃথিবীতে আলো দেয়নি সূর্যদেবতা টাইটান, | nullus adhuc mundo praebebat lumina Titan, |  |
| তখনো চিকন শিং নতুন করত না চাঁদের দেবী ফিবি, | nec nova crescendo reparabat cornua Phoebe, |  |
| তখনো পৃথিবী তার নিজের ওজনে শূন্যের মাঝখানে | nec circumfuso pendebat in aere tellus |  |
| ঝুলতে শুরু করেনি, আর সমুদ্রের দেবী আম্ফিত্রিতি | ponderibus librata suis, nec bracchia longo |  |
| দিঘল দুই বাহু দিয়ে জড়িয়ে ধরেনি মাটির সীমান্ত; | margine terrarum porrexerat Amphitrite; |  |
| জল স্থল আর অন্তরীক্ষ যে তখন ছিল না তা নয়, | utque aer, tellus illic et pontus et aether. |  |
| তবে সেই স্থল ছিল অস্থির, জল সাঁতারের অযোগ্য, | Sic erat instabilis tellus, innabilis unda, |  |
| বাতাস অন্ধকার; এক রূপে সুস্থির থাকত না কিছু, | lucis egens aer: nulli sua forma manebat, |  |
| সবার সাথে সবার দ্বন্দ্ব, কারণ এক দেহের ভিতরে | obstabatque aliis aliud, quia corpore in uno |  |
| যুদ্ধ করত গরমের সাথে শীত আর শুষ্কের সাথে | frigida pugnabant calidis, umentia siccis, |  |
| সিক্ত, কঠিনের সাথে কোমল আর হালকার সাথে ভারী। | mollia cum duris, sine pondere habentia pondus. |  |
| | | |
| এক খোদা (বা আরো দয়াল প্রকৃতি) এই বিরোধ মিটান, | Hanc deus et melior litem natura diremit. | 21 |
| তিনি ছিন্ন করেন আকাশ থেকে মাটি, মাটি থেকে সাগর; | Nam caelo terras et terris abscidit undas, |  |
| ঘন বায়ুমণ্ডল থেকে তরল মহাশূন্য আলাদা করেন। | et liquidum spisso secrevit ab aere caelum. |  |
| যখন অন্ধ স্তূপ থেকে সব ভাগ করে মুক্ত করেন, | Quae postquam evolvit caecoque exemit acervo, |  |
| যার যার জায়গায় সবাই সুরেলা বন্ধনে বাঁধা পড়ে। | dissociata locis concordi pace ligavit. |  |
| আগুনের মত ভরহীন উপাদান ইথার সবার আগে | Ignea convexi vis et sine pondere caeli |  |
| এক লাফে স্বর্গের ছাদে অধিকার করে তার জমি, | emicuit summaque locum sibi fecit in arce: |  |
| ওজনের দিক দিয়ে তার ঠিক পরে আসে বাতাসের স্থান, | proximus est aer illi levitate locoque: |  |
| মাটি এর চেয়ে ভারী, সে অন্য সব ভারী উপাদান নিয়ে | densior his tellus, elementaque grandia traxit |  |
| নিজের ভারে সবার কেন্দ্রে আসন গাড়ে, সব-ঘেরা জল | et pressa est gravitate sua: circumfluus umor |  |
| শেষ জায়গাটা নিয়ে আবৃত করে রাখে কঠিন ভূমিকে। | ultima possedit solidumque coercuit orbem. |  |
| | | |
| এভাবে সেই দেবতা (তা তিনি যিনিই হন) জগৎ সাজান, | Sic ubi dispositam quisquis fuit ille deorum | 32 |
| কেয়সকে ভাগ করে অংশগুলোর বিবাদ দূর করেন, | congeriem secuit sectamque in membra redegit, |  |
| এবং সবার আগে এক বিশাল গোলক হিসেবে পৃথিবী | principio terram, ne non aequalis ab omni |  |
| তৈরি করেন যাতে তা এক রকম হয় সব দিক থেকে। | parte foret, magni speciem glomeravit in orbis. |  |
| তারপর জলকে আদেশ করেন ছড়িয়ে পড়তে, ঝড়ের | Tum freta diffudit rapidisque tumescere ventis |  |
| টানে উত্তাল হয়ে পৃথিবীর সব তীর ঘেরাও করতে। | iussit et ambitae circumdare litora terrae. |  |
| তিনি বানিয়েছিলেন সচল ঝরনা, নিশ্চল দিঘি হ্রদ, | Addidit et fontes et stagna inmensa lacusque |  |
| দু-পারের পাহারায় বেঁধেছিলেন নিম্নগামী সব নদী, | fluminaque obliquis cinxit declivia ripis, |  |
| শত ধারায় ধাবিত যাদের পানি কিছুটা শুষে নেয় মাটি, | quae, diversa locis, partim sorbentur ab ipsa, |  |
| আর বাকিটা সাগরে ছুটে যায়, আরো বিপুল আরো অবাধ | in mare perveniunt partim campoque recepta |  |
| বন্যার সাথে মিলে তীরের বদলে ভেঙে পড়ে উপকূলে। | liberioris aquae pro ripis litora pulsant. |  |
| তারপর তার আদেশে সমতল ছড়ায়, অবতল নামে, | Iussit et extendi campos, subsidere valles, |  |
| পাথুরে পাহাড় জেগে উঠে, অরণ্য পরে সবুজ কাপড়। | fronde tegi silvas, lapidosos surgere montes. |  |
| আর আকাশ যেমন ডানে দুই বামে দুই ভাগে বিভক্ত | Utque duae dextra caelum totidemque sinistra |  |
| আর মাঝখানে পঞ্চম অঞ্চল সবার চেয়ে তপ্ত, | parte secant zonae, quinta est ardentior illis, |  |
| তেমনি পরিবৃত পৃথিবীর তল পাঁচ ভাগে ভাগ হয় | sic onus inclusum numero distinxit eodem |  |
| স্রষ্টার ইচ্ছায়—ভূত্বকে সমান সংখ্যক রেখা আঁকা। | cura dei, totidemque plagae tellure premuntur. |  |
| কেন্দ্রের এলাকায় থাকা যায় না বেশি গরমের কারণে, | Quarum quae media est, non est habitabilis aestu: |  |
| দুই মেরুতে তুষার, বাকি দুই ভাগ তিনি মাঝখানে রেখে | nix tegit alta duas: totidem inter utrumque locavit |  |
| নাতিশীতোষ্ণ জলবায়ু দেন তাপের সাথে শীত মিশিয়ে। | temperiemque dedit mixta cum frigore flamma. |  |
| | | |
| এদের উপর চাপিয়েছিলেন বায়ুপথ, যা অগ্নিময় | Inminet his aer. Qui quanto est pondere terrae, | 52 |
| ইথারের চেয়ে তত ভারী পানির চাইতে মাটি যত ভারী। | pondere aquae levior tanto est onerosior igni. |  |
| সেখানেই তিনি জড়ো করেন মেঘ, জমা করেন কুহেলিকা, | Illic et nebulas, illic consistere nubes |  |
| মানুষের অন্তর থরথর কাঁপাতে লেলিয়ে দেন বজ্র, | iussit et humanas motura tonitrua mentes |  |
| আর বজ্র থেকে বিদ্যুৎ খসাতে লাগিয়ে দেন বাতাস। | et cum fulminibus facientes fulgura ventos. |  |
| বিশ্বকর্মা আকাশের সবখানে সব বায়ুকে ঘুরার | His quoque non passim mundi fabricator habendum |  |
| অনুমতি দেন নাই, এদেরকে এখনও ঠেকানো যায় না | aera permisit: vix nunc obsistitur illis, |  |
| যখন একেক এলাকায় রাজত্ব করা একেক বাতাস | cum sua quisque regant diverso flamina tractu, |  |
| বিশ্ব প্রায় ধ্বংস করে দেয়, ভায়ে ভায়ে এমন বিবাদ। | quin lanient mundum: tanta est discordia fratrum. |  |
| ইউরাস গিয়েছিল পূর্বে উষার দেশে, যেখানে ইরানি | Eurus ad Auroram Nabataeaque regna recessit |  |
| ও আরব পাহাড়ের চূড়ায় রশ্মি হানে ভোরের সূর্য; | Persidaque et radiis iuga subdita matutinis; |  |
| সূর্যাস্তের আলো যে সকল উপকূল আলোকিত করে | vesper et occiduo quae litora sole tepescunt, |  |
| সেই পশ্চিমে জেফারের রাজ্য; স্কিথিয়া আর উত্তর | proxima sunt Zephyro: Scythiam septemque triones |  |
| মেরু অধিকার করে বোরিয়াস; তার বিপরীতে দক্ষিণে | horrifer invasit Boreas: contraria tellus |  |
| অবিরাম কুয়াশা ও বৃষ্টিতে ভেজা দেশে থাকে অস্টার। | nubibus adsiduis pluviaque madescit ab Austro. |  |
| এই চার বাতাসের উপরে তিনি বসিয়েছিলেন ইথার, | Haec super inposuit liquidum et gravitate carentem |  |
| ওজনহীন তরল, যার মাঝে কোনো পার্থিব কণা নাই। | aethera nec quicquam terrenae faecis habentem. |  |
| | | |
| এভাবে নির্দিষ্ট সীমানায় সব বাঁধতে না বাঁধতেই | Vix ita limitibus dissaepserat omnia certis, | 69 |
| এতকাল অন্ধকারের অন্তরালে লুকিয়ে থাকা সব | cum, quae pressa diu massa latuere sub illa, |  |
| তারা সারা মহাকাশ জুড়ে আগুনের মতো জ্বলা শুরু করে। | sidera coeperunt toto effervescere caelo. |  |
| আর প্রতিটি অঞ্চলকে যার যার যোগ্য জীবন দিতে | Neu regio foret ulla suis animalibus orba, |  |
| দেবতাদের রূপ ও তারামণ্ডল বাসা বাঁধে মহাকাশে, | astra tenent caeleste solum formaeque deorum, |  |
| ঝকঝকানো মাছের জন্য জায়গা করে দেয় সমুদ্র, | cesserunt nitidis habitandae piscibus undae, |  |
| মাটি পায় পশুদের, আর অবাধ বাতাস হয় পাখিদের। | terra feras cepit, volucres agitabilis aer. |  |
| এদের চেয়ে পবিত্র, মনের দিক দিয়ে আরো মহীয়ান | Sanctius his animal mentisque capacius altae |  |
| কোনো প্রাণী ছিল না যে বাকি সবাইকে শাসন করতে পারে। | deerat adhuc et quod dominari in cetera posset. |  |
| তখন মানুষ হলো—হয় নিজের দৈব বীজ থেকে তাকে | Natus homo est, sive hunc divino semine fecit |  |
| বানিয়েছিলেন আরো ভালো এই বিশ্বের বিধানকর্তা, | ille opifex rerum, mundi melioris origo, |  |
| অথবা এই মাত্র ইথার থেকে আলাদা হওয়া মৃদু মাটি | sive recens tellus seductaque nuper ab alto |  |
| (যার তখনো আত্মীয়তা রয়ে গেছে মহাশূন্যের সাথে) | aethere cognati retinebat semina caeli; |  |
| হাতে নিয়ে তার সাথে পানি মিশিয়ে ইয়াপিটাসের পুত্র | quam satus Iapeto mixtam pluvialibus undis |  |
| বানিয়েছিলেন সর্বনিয়ন্ত্রক দেবতাদের আদল। | finxit in effigiem moderantum cuncta deorum. |  |
| যেখানে অন্য সব প্রাণী মাথা ঝুলিয়ে রাখে মাটির দিকে | Pronaque cum spectent animalia cetera terram, |  |
| সেখানে মানুষকে দিয়েছেন উন্নত শির, যা আকাশে তুলে | os homini sublime dedit, caelumque videre |  |
| চোখ তাক করতে বলেছেন তারাখচিত খিলানের দিকে। | iussit et erectos ad sidera tollere vultus. |  |
| যে মাটি অতীতে ছিল রূপহীন রসহীন তাকে এইভাবে | Sic, modo quae fuerat rudis et sine imagine, tellus |  |
| ঢালাই করে দিয়েছিলেন মানুষের নতুন অচিন রূপ। | induit ignotas hominum conversa figuras. |  |
| | | |
| স্বর্ণযুগ ছিল প্রথম যুগ, যখন আইন না থাকলেও | Aurea prima sata est aetas, quae vindice nullo, | 89 |
| মানুষ স্বাধীন ইচ্ছায় নীতি ছাড়া ন্যায়ের পূজা করত। | sponte sua, sine lege fidem rectumque colebat. |  |
| শাস্তির ভয় ছিল না, মানুষ পড়ত না ব্রঞ্জে খোদাই | Poena metusque aberant, nec verba minantia fixo |  |
| করা হুমকির বাণী, বিচারকের সামনে হাঁটু গাড়ত না | aere legebantur, nec supplex turba timebat |  |
| শঙ্কিত অপরাধী, জজ ছাড়াই সবাই ছিল নিরাপদ। | iudicis ora sui, sed erant sine vindice tuti. |  |
| পাহাড়ের ভরা গাল থেকে কেটে নিয়ে কোনো পাইন গাছ | Nondum caesa suis, peregrinum ut viseret orbem, |  |
| বিদেশের উদ্দেশে ভাসানো হয়নি সাগরের তরঙ্গে, | montibus in liquidas pinus descenderat undas, |  |
| মানুষ তার নিজের কূল ছাড়া আর কোনো তীর চিনত না। | nullaque mortales praeter sua litora norant. |  |
| গভীর পরিখা দিয়ে ঘেরাও করা হয়নি কোনো শহর, | Nondum praecipites cingebant oppida fossae; |  |
